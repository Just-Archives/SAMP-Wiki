Description:

Deze callback word opgeroepen wanneer een speler één van de button op de met ShowPlayerDialog gecreeërde dialog drukt.


Parameters:
(playerid, dialogid, response, listitem, inputtext[])
playeridDe id van de speler waarvoor de dialog verscheen..
dialogidDe id van de dialog die voor de speler verscheen, de dialogid word bepaald in ShowPlayerDialog.
response1 voor de eerste button 0 voor de tweede button
listitemDe ID van de item in de lijst geselecteerd door de speler.
inputtext[]De tekst ingevoerd in het invul vak door de speler of de geselecteerde item tekst in de lijst.


public OnDialogResponse(playerid, dialogid, response, listitem, inputtext[])
{
    if(!IsPlayerAdmin(playerid)) return 0; // dit is enkel voor admins
    if(dialogid == 1)
    {
        if(!response) SendClientMessage(playerid, 0xFFFFFFFF, "Jij weigerde!");
        new message[196];
        format(message, sizeof(message), "Je antwoordde: %s", inputtext);
        SendClientMessage(playerid, 0xFFFFFFFF, message);
        return 1;
    }
    return 0;
}

Related Functions

The following functions might be useful, as they're related to this callback in one way or another.

  • ShowPlayerDialog: Show a dialog to a player.
