Description:

Cette callback est appelée lorsqu'un joueur sélectionne un article depuis un menu.


Paramètres:
(playerid, row)
playeridL'ID du joueur qui a sélectionné le menu.
rowL'ID de la ligne qui a été sélectionnée.


Retourne:

Cette callback ne retourne rien, mais doit retourner quelque chose! Regardez ICI pour plus d'informations.


Note

L'ID du menu choisi n'est pas un paramètre de cette callback. GetPlayerMenu doit être utilisé pour déterminer quel menu le joueur a-t-il sélectionné.


Example Usage: 

new Menu:monMenu;
 
public OnGameModeInit()
{
    monMenu= CreateMenu("Menu test", 1, 50.0, 180.0, 200.0, 200.0);
    AddMenuItem(myMenu, 0, "Article 1");
    AddMenuItem(myMenu, 0, "Article 2");
    return 1;
}
 
public OnPlayerSelectedMenuRow(playerid, row)
{
    if(GetPlayerMenu(playerid) == myMenu)
    {
        switch(row)
        {
            case 0: print("Le joueur d'ID %d a sélectionné l'item 1.", playerid);
            case 1: print("Le joueur d'ID %d a sélectionné l'item 2.", playerid);
        }
    }
    return 1;
}
