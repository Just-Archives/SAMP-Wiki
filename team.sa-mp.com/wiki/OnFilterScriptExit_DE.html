Description:

Dieses Callback wird aufgerufen, wenn ein Filterscript beendet wird.


Dieses Callback hat keine Parameter.


Dieses Callback benutzt den Return-Wert nicht.


public OnFilterScriptExit()
{
    print("Filterscript beendet.");
    return 1;
}

Verwandte Callbacks

Die folgenden Callbacks könnten nützlich sein, da sie auf eine Weise mit diesem Callback zu tun haben.

  • OnFilterScriptInit: Wird aufgerufen, wenn ein Filterscript geladen wird.
  • OnGameModeInit: Wird aufgerufen, wenn ein Gamemode gestartet wird.
  • OnGameModeExit: Wird aufgerufen, wenn der Gamemode beendet wird.
