클라이언트

[edit]

"San Andreas cannot be found" 오류가 뜨는 경우

  • San Andreas Multi-player 는 절대 혼자서 실행할 수가 없습니다! San Andreas EU/US v1.0 버전이 있어야 실행 할 수 있습니다. v2.0, Steam, Direct2Drive 등의 버전은 쓸 수 없습니다.
[edit]

SA:MP 브라우져에 서버가 안뜨는 경우

First off, ensure you are following the procedures set out in the Quick-start guide. If you have followed what it says, and still cannot see any servers, you must allow SA:MP access through your firewall. Unfortunately, due to the large amount of available firewall software, we can not offer further support on this - we suggest looking at the manufacturers website or trying a Google search. Also make sure you have the latest SA:MP version!

[edit]

SA:MP 대신 싱글플레이가 실행되는 경우

YOU ARE NOT SUPPOSED TO SEE THE SINGLE PLAYER OPTIONS (NEW GAME, LOAD GAME, ETC) - SA:MP SHOULD LOAD BY ITSELF AND NOT PRESENT THESE OPTIONS. IF YOU SEE "NEW GAME" SINGLE PLAYER HAS LOADED, NOT SAN ANDREAS MULTIPLAYER.

Single Player can load for 2 reasons - you have installed SA:MP to the wrong folder or you have the wrong version of San Andreas. If you have the wrong version it's easy to fix. Download this patch that downgrades it to the right version. Click here to download the patch.

[edit]

서버에 접속할 때 "Unacceptable NickName" 라는 오류가 뜰 경우

Ensure you are not using any disallowed characters in your name (use 0-9, a-z, [], (), $, @, ., = only), and that your name is no longer than 24 characters long. This also could be caused when a player is on a server with the same name as you or if you try connecting to the server immediately after timing out.

[edit]

화면이 "Connecting to IP:Port..." 에서 멈추는 경우

The server could be offline, or if you cant connect to any server, disable your firewall and see if it works. If it does you must set up your firewall properly - go to its website to find out how. It could also be that you have an old version of SA-MP, download the latest version from the SA-MP download page.

[edit]

GTA: San Andreas 를 수정한 후, SA:MP 가 로딩되지 않는 경우

If it wont load then remove your mods. More information why mods can cause issues with your game can be found here

[edit]

SA:MP로 GTA를 시작할 때 시작되지 않는 경우

Delete the gta_sa.set file from your userfiles folder and make sure you don't have any cheats/mods.

[edit]

서버

[edit]

서버가 계속 시작할 때 꺼질 경우

  • 보통은 server.cfg 파일에 문제가 있는 경우입니다. server_log.txt 를 확인하시거나 crashinfo.txt 파일을 이용해 원인을 파악하십시오.
[edit]

서버가 작동하지 않아 방화벽을 해제해도 작동하지 않는 경우

  • 전송 포트를 열어야 합니다! PF Port Checker 프로그램으로 열 수 있습니다. (다운 로드 : www.portforward.com)
  • 그래도 포트가 열리지 않는다면, 라우터 또는 공유기 등의 인터넷 장비에서 포트를 열어야 합니다.
[edit]

플레이어들이 가능한 이름이지만 "Unacceptable NickName" 오류를 계속 받는 경우

If you are sure you use an acceptable name and the server runs on Windows then try to change the compatibility option of the samp-server.exe to Windows 98 and it should be fixed after a server restart.

Windows servers with high uptime may also cause this issue to occur. This has been noticed of around 50 days of server uptime. To resolve it, a reboot is required.
