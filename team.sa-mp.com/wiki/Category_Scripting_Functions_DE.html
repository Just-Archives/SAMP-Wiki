Diese Seite beinhaltet eine Übersicht der unzähligen Funktionen des San Andreas Multiplayers. Alle Funktionen haben eine eigene Seite mit erweiterten Informationen zur Funktion, den benötigten Parametern, einer möglichen Verwendung und anderen Funktionen, die Dich eventuell interessieren könnten.

Mit jeder neuen SA:MP Version erweitert sich die Funktionsliste. Die Server Plugins können die Funktionsliste von SA:MP auch erweitern, klicke auf den Link, um mehr Informationen zu den Server Plugins zu erhalten. Im Moment sind 273 native Functions verfügbar.

Für eine alphabetisch geordnete Liste, bitte nach unten Scrollen.

[edit]

Kategorien

[edit]

Menüs

  • CreateMenu - Erstellt ein Menü.
  • AddMenuItem - Fügt einen Eintrag dem angegebenen Menü hinzu.
  • SetMenuColumnHeader - Legt die Spaltenbeschriftung fest.
  • DestroyMenu - Löscht ein erstelltes Menü.
  • ShowMenuForPlayer - Zeigt einem Spieler das erstellte Menü an.
  • HideMenuForPlayer - Lässt das Menü für einen Spieler verschwinden.
  • GetPlayerMenu - Gibt die ID des Menüs zurück, welches sich auf dem Spielerbildschrim befindet.
