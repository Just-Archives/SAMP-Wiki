RepairVehicle

Deze functie werd toegevoegd in SA-MP 0.3a en werkt niet in eerdere versies!


Beschrijving:

Herstelt de visuele schade (bulten, deuken, krassen, wielen en de werkelijke health) van een voertuig.


Parameters:
(vehicleid)
vehicleidDe ID van het voertuig dat je wil herstellen.


Retourneert:

  • 1: De actie werd succesvol uitgevoerd.
  • 0: Het uitvoeren van de actie is mislukt. Het opgegeven voertuig bestaat niet.


Voorbeeld: 

public OnPlayerCommandText(playerid, cmdtext[])
{
    if (!strcmp("/repair", cmdtext))
    {
        if(!IsPlayerInAnyVehicle(playerid)) return SendClientMessage(playerid, 0xFFFFFFFF, "Je bent niet in een voertuig!");
        RepairVehicle(GetPlayerVehicleID(playerid));
        SendClientMessage(playerid, 0xFFFFFFFF, "Je voertuig is succesvol hersteld!");
        return 1;
    }
}

Extra Functies

De volgende functies zijn mogelijk ook bruikbaar, aangezien ze op een manier erbij horen.

  • SetVehicleHealth: Wijzigt de health van een voertuig.
  • GetVehicleHealth: Verkrijg de health van een voertuig.
