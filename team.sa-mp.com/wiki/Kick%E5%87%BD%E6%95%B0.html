这个函数能踢除当前正在你服务器里的某个玩家。


参数表:
(playerid)
playerid想要踢除的 玩家ID 。
返回此函数不返回任何特定的值。


public OnPlayerCommandText( playerid, cmdtext[] )
{
    if( strcmp( cmdtext, "/kickme", true ) == 0 )
    {
        // 谁输入这指令谁就会被踢出服务器
        Kick( playerid );
        return 1;
    }
}

有关联的函数

以下函数可能会对你有所帮助，他们可能跟这个函数有直接关系或间接关系。

  • Ban: 封禁服务器里的某个玩家。
  • BanEx: 用指定理由封禁一个玩家。
