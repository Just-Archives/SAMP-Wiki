NPC:OnNPCExitVehicle

Этот обратный вызов добавлен в SA-MP 0.3 и не работает в более ранних версиях!


Описание:

Вызывается, когда NPC выходит из автомобиля.


Параметры:
(Не имеет параметров)


Возвращаемые значения:

Эта функция не возвращает определённого значения


Пример использования: 

public OnNPCExitVehicle()
{
    print("NPC вышел из автомобиля");
    return 1;
}

Связанные функции

Следующие функции могут быть полезны, т.к. они так или иначе связаны с текущей функцией.
