IsPlayerAdmin

Description:

Bij gebruik van deze functie kan je opvragen of de speler ingelogd is als RCON admin door gebruik van de in-game /rcon commando's.


Parameters:
(playerid)
playeridHet ID van de speler die je wilt controleren.


Return Values:

Stuurt 'true' terug wanneer het een admin is en 'false' wanneer het geen admin is.


public OnPlayerSpawn(playerid, cmdtext[])
{
   if(IsPlayerAdmin(playerid))
   {
      SendClientMessageToAll(0xDEEE20FF, "Een admin is gespawned.");
   }
   return 1;
}
