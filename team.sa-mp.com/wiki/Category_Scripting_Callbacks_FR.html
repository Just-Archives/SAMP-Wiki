Cette page contient un aperçu de toute les callbacks disponible dans SA:MP Les "Callback" sont appelés dans le script par le serveur lorsque quelque chose d'important se passe (Joueur qui meurt/se déconnecte/entre dans un véhicule/ré-apparait etc...). La plupart des Callbacks ont le paramètre "playerid" qui désigne l'ID d'un joueur, mais pas toute.


'Important: Quand vous utilisez une callback dans un filter script, assurez vous de retourner 1 (return 1). Si vous ne retournez pas 1 mais 0, Les autres callbacks d'autres filterscript ne seront pas executées. Retourner une valeur n'a pas d'importance dans OnFilterScriptInit et OnFilterScriptExit. Dans OnPlayerCommandText, retourner 1 dira a la callback que la commande a été trouvée. Retourner 0 dira que la commande n'a pas été trouvée, et un message d'erreur (prédéfini) sera envoyé au joueur.
