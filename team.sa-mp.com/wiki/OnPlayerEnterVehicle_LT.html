Description:

Šis callback'as iškviečiamas, kai žaidėjas pradeda įlipimo į transporto priemonę veiksmą.


Parametrai:
(playerid, vehicleid, ispassenger)
playeridŽaidėjo ID, kuris pradėjo įlipimo veiksmą.
vehicleidTransporto priemonės ID, į kurią žaaidėjas bando įlipti.
ispassenger0 - jeigu žaidėjas lipa į vairuotoją vietą. 1 - jeigu lipa į keleivio.


Šis callback'as neturi grąžinamos reikšmės.


Note

Šiame callback'e žaidėjas negali būti transporto priemonėje. 


public OnPlayerEnterVehicle(playerid, vehicleid, ispassenger)
{
    new stringas[128];
    format(stringas, sizeof(stringas), "Tu bandai įlipti į transporto priemonę, kurios ID: %i.", vehicleid);
    SendClientMessage(playerid, 0xFFFFFFFF, string);
    return 1;
}
