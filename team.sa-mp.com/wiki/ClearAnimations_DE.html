Description:

Hält alle aktuell vom Spieler ausgeführten Animationen an.


Parameter:
(playerid)
playeridDie ID des Spielers, dessen Animationen gestoppr werden sollen.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


public OnPlayerCommandText(playerid, cmdtext[])
{
    if(!strcmp(cmdtext, "/animsoff", true))
    {
        ClearAnimations(playerid);
        return 1;
    }
    return 0;
}


Diese Funktion wurde in SA:MP 0.3b verändert. Die Dokumentation für die neue Version befindet sich unten.


Parameter:
(playerid, forcesync)
playeridDie ID des Spielers, dessen Animationen gestoppr werden sollen.
forcesync(optional) Bestimmt, ob der Abbruch der Animationen mit allen Spielern in der Nähe synchronisiert werden soll.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.
