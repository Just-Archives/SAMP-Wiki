Hauptseite Basics Funktionen Callbacks Plugins (engl.)

| Delete3DTextLabel wurde erst in SA-MP 0.3a hinzugefügt und funktioniert nicht in älteren Versionen.


Beschreibung:

  • Löscht ein erstelltes 3DTextLabel.


Parameter:

    (Text3D:id)
Text3D:idDie ID vom 3DTextLabel der gelöscht werden soll.


Rückgabe Werte:

  • 1 falls die Funktion erfolgreich ausgeführt wurde.
  • 0 falls die ID vom 3DTextLabel nicht existiert.


| Pawno Beispiel: 

new Text3D:MeinLabel;
 
OnGameModeInit()
{
    Create3DTextLabel("Mein eigenes Label", 0xFFFFFFFF, 0, 0, 0, 15.0, -1, 0);
    return 1;
}
 
OnGameModeExit()
{
    Delete3DTextLabel(MeinLabel);
    return 1;
}
