AddStaticVehicle函数

Description:

你能將交通工具新增至地圖中. 這個函數只能放在 OnGameModeInit 以及 OnFilterScriptInit 回调中. 如果你是想在遊戲過程中產生車輛 CreateVehicle 使用函數才是正確的.


Parameters:
(車輛模組ID, Float:初始點X, Float:初始點Y, Float:初始點Z, Float:角度, 顏色1, 顏色2)
車輛模組ID 請參考 Model ID 內的車輛模組ID.
Float:初始點X車輛的座標X.
Float:初始點Y車輛的座標Y.
Float:初始點Z車輛的座標Z.
Float:角度車輛的角度.
顏色1主要顏色 color ID.
顏色2次要顏色 color ID.


Return Values:

創造的車輛VID.


public OnGameModeInit( )
{
    // 新增一台 Hydra 在遊戲中
    AddStaticVehicle(520, 2109.1763, 1503.0453, 32.2887, 82.2873, 0, 1);
 
    return 1;
}
