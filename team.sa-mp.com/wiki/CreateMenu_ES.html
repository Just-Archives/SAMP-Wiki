Descripción:

Crea un menú.


Parámetros:
(title[], columns, Float:x, Float:y, Float:col1width, Float:col2width)
title[]El título para el nuevo menú.
columnsCuántas columnas podrá tener el menú.
Float:xLa posición X del menú (límite de 640x460 - 0 pondrá al menú en el extremo izquierdo).
Float:yLa posición Y del menú (límite de 640x460 - 0 pondrá al menú en la parte superior).
Float:col1widthEl ancho de la primera columna.
Float:col2widthEl ancho de la segunda columna.


Retornos/Devoluciones:

La ID del menú que creó, retorna -1 si falló (no se puede crear más menús)



Nota
Importante

Hay un límite de 12 ítems por menú, y un límite de 128 menús en total.



Notas

  • Esta función simplemente CREA al menú - ShowMenuForPlayer debe ser usado para mostrarlo a un jugador.
  • Usted sólo puede crear y acceder a 2 columnas (0 & 1).
  • Si la longitud del título es igual o mayor a 32 caracteres, el título se truncará a 30 caracteres. 


new Menu:examplemenu;
 
public OnGameModeInit()
{
    examplemenu = CreateMenu("Tu menu", 2, 200.0, 100.0, 150.0, 150.0);
    return 1;
}
[edit]

Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • AddMenuItem: Add an item to a menu.
  • SetMenuColumnHeader: Set the header for one of the columns in a menu.
  • DestroyMenu: Destroy a menu.
  • ShowMenuForPlayer: Show a menu for a player.
  • HideMenuForPlayer: Hide a menu for a player.


[edit]

Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • OnPlayerSelectedMenuRow: Called when a player selected a row in a menu.
  • OnPlayerExitedMenu: Called when a player exits a menu.
