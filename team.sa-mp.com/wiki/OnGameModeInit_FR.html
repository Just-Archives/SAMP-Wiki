Description:

Cette callback est appelée lorsque le gamemode est chargé.


Cette callback n'a pas de paramètres.


Retourne:

Cette callback ne retourne rien, mais doit retourner quelque chose! Regardez ICI pour plus d'informations.


public OnGameModeInit()
{
    print("Gamemode chargé!");
    return 1;
}
[edit]

Callback Relatives

Les Callbacks ci dessous sont indirectement ou directement liées à cette Callback.

  • OnGameModeExit: Appelée lorsqu'un gamemode se ferme.
  • OnFilterScriptInit: Appelée lorsqu'un filtescript est chargé.
  • OnFilterScriptExit: Appelée lorsqu'un filterscript se ferme.
