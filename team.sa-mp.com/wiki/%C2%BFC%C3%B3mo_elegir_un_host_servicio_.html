¿Cómo elegir un host/servicio?

From SA-MP Wiki

Jump to: navigation, search

Esta guía es para las personas que piensan en comprar un hosting, y no saben cómo elegir; tendrás que usar tu propio criterio para elegir tu servicio adecuado y llevar adelante tu servidor.


¿Cuanta Banda Ancha se recomienda?

1 servidor = 50 GB/Mes

1 o más servidores = 100 GB/Mes


¿VPS (virtual private server - servidor virtual privado) o dedicado?

VPS: Un VPS es un método de particionar (partir) un servidor físico en varios servidores de tal forma que todo funcione como si se estuviese ejecutando en una única máquina. (cita de Wikipedia)

Dedicado: Un servidor dedicado es un ordenador comprado o arrendado que se utiliza para prestar servicios dedicados, generalmente relacionados con el alojamiento web y otros servicios en red. (cita de Wikipedia)


¿Windows o Linux?

Utiliza Linux si tienes algún tipo de conexión con alguna página web (por ejemplo, si utilizas MySQL), además Linux es más seguro.

Utiliza Windows si tu servidor consumo poca memoria RAM (además podrías poner otro servidor), y si trabajas en el manejo de datos utilizando Dini, Y_INI o algún otro guardado de datos nativos a SA-MP.

Los servidores en Linux utilizan plugins con la extensión .so, Windows utiliza la extensión .dll.


¿Cuánta memoria RAM?

256 MB = Ideal para servidores livianos. Tu servidor podría generar un crash si utilizas plugins muy pesados (como MapAndreas).

512 MB = Ideal para servidores que no necesita conexiones con otros servidores.

1,0 GB = Ideal para servidores que utilizan MySQL.

2,0 GB = Ideal para servidores que utilizan MySQL y conexiones con otros servidores.


¿Qué CPU se necesita?

Servidores RolePlay: Intel Core 2 Extreme - Intel i7

Servidores Sunt: Intel Core 2 Duo, AMD ATLHON

Servidores Deatmatch: Intel ATOM
