Description:

Esta callback é chamado quando um jogador digita um comando no chat, Ex.: /ajuda.


Parâmetros:
(playerid, cmdtext[])
playeridID do jogador que digitou o comando.
cmdtext[]O comando que sera executado (incluindo a barra).


Valores de retorno:

Retorna 1 se o comando foi processado, caso contrário 0.


public OnPlayerCommandText(playerid, cmdtext[])
{
    if(!strcmp(cmdtext, "/ajuda", true))
    {
        SendClientMessage(playerid, 0xFFFFFFFF, "SERVER: Este é comando /ajuda!");
        return 1;
        // Retornando 1 informa ao servidor que o comando foi processado.
    }
    return 0;
    // Retornando 0 informa ao servidor que o comando não foi processado por este script.
    // OnPlayerCommandText será chamado em outros scripts até que um retorne 1.
    // Caso nenhum script retornar 1, sera exibida a mensagem 'SERVER: Unknown Command'.
}
