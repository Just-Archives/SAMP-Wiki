AttachTrailerToVehicle

Описание:

Присоединяет прицеп к транспорту



Важное
Примечание

Это будет работать только если оба транспортных средства видны игроку (для проверки используйте IsVehicleStreamedIn).


Параметры:
(trailerid, vehicleid)
traileridID прицепа, который будет присоединён
vehicleidID транспорта, к которому будет присоединён прицеп


Возвращаемые значения:

Эта функция всегда возвращает 1, даже если trailerid и/или vehicleid не существует.


Пример использования: 

public OnGameModeInit()
{
    new vehicleid = CreateVehicle(403, 2424.8999, 2791.7, 11.5, 270.0, 34, 25, 600, 0);
    new trailerid = CreateVehicle(584, 251.4276, 1396.1947, 11.1916, 271.8214, 1, 1, 600, 0);
    AttachTrailerToVehicle(trailerid, vehicleid);
    return 1;
}


[edit]

Связанные функции

Следующие функции могут быть полезны, т.к. они так или иначе связаны с текущей функцией.

  • DetachTrailerFromVehicle: Отсоединяет прицеп от транспорта.
  • IsTrailerAttachedToVehicle: Проверяет, присоединен ли прицеп к транспорту.
  • GetVehicleTrailer: Проверяет, какой прицеп тащит транспорт.
