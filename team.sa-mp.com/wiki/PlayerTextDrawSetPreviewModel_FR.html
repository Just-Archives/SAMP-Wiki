Cette function a été ajoutée dans SA-MP 0.3x et ne marchera pas dans les versions antérieures


Description:

Applique un sprite en 2D sur un PlayerTextDraw.


Paramètres:
(playerid, PlayerText:text, modelindex)
playeridLe joueur dont le PlayerTextDraw va être modifié.
textL'ID du TextDraw qui va affiché le sprite.
modelindexL'ID du sprite à afficher.


Retourne:

  • 1: La fonction a bien été exécutée. Si un sprite invalide est inseré un point d'intérogation jaune et noir s'affichera.
  • 0: La fonction n'a pas pu être exécutée. Le joueur ou l'ID du PlayerTextDraw n'existe pas.


Example Usage: 

new PlayerText:textdraw;
 
public OnPlayerConnect(playerid)
{
    textdraw = CreatePlayerTextDraw(playerid, 320.0, 240.0, "_");
    PlayerTextDrawFont(playerid, textdraw, TEXT_DRAW_FONT_MODEL_PREVIEW);
    PlayerTextDrawUseBox(playerid, textdraw, 1);
    PlayerTextDrawBoxColor(playerid, textdraw, 0x000000FF);
    PlayerTextDrawTextSize(playerid, textdraw, 40.0, 40.0);
    PlayerTextDrawSetPreviewModel(playerid, textdraw, 411); // Montre une infernus Infernus (sprite 411)
    //PlayerTextDrawSetPreviewModel(playerid, textdraw, 0); //Montre le skin 0 (CJ)
    //PlayerTextDrawSetPreviewModel(playerid, textdraw, 18646); //Montre le sprite 18646 ((Objet) lumière de police)
    PlayerTextDrawShow(playerid, textdraw);
    return 1;
}

Note
Importante

Le PlayerTextDraw doit utilisé la police (font) TEXT_DRAW_FONT_MODEL_PREVIEW pour pouvoir afficher le sprite.



[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • PlayerTextDrawSetPreviewRot FR: Modifie la rotation du sprite affiché sur le PlayerTextDraw
  • PlayerTextDrawSetPreviewVehCol FR: Modifie la couleur du vehicule affiché sur le PlayerTextDraw
  • PlayerTextDrawFont FR: Modifie la police d'écriture du PlayerTextDraw

Callbacks Relatives

Les Callbacks suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • OnPlayerClickPlayerTextDraw FR: Est appelé quand un joueur appuie sur un autre joueur affiché dans la liste de joueurs connecté (TAB)
