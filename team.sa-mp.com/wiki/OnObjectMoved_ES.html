Description:

Este callback es llamado cuando un objeto termina de moverse.


Parámetros:
(objectid)
objectidID del objeto que fue movido.
RetornoEsta función no retorna un valor específico


Nota

SetObjectPos no funciona si es usado en este callback. Para solucionarlo borre y vuelva a crear el objeto.


Ejemplo: 

public OnObjectMoved(objectid)
{
    printf("El objeto %i fue movido.",objectid);
    return 1;
}

Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • MoveObject Mueve un objeto.
  • StopObject Para un objeto que este en movimiento.


Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • OnPlayerObjectMoved Llamado cuando el objeto de un jugador finaliza su movimiento.
