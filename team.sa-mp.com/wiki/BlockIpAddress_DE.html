Hauptseite Basics Funktionen Callbacks Plugins (engl.)


| BlockIpAdress wurde erst in SA-MP 0.3z R2-2 hinzugefügt und funktioniert nicht in älteren Versionen.


Beschreibung:

  • Blockiert eine IP-Adresse auf Zeit.


| Hinweise:
Spieler die versuchen mit einer blockierten IP-Adresse auf den Server zu verbinden, erhalten die Nachricht "You are banned from this server.". Spieler die während der Einrichtung der Blockierung online sind erhalten die selbe Nachricht.


Parameter:

    (ip_address[], timems)
ip_addressDie IP-Adresse die blockiert werden soll.
timemsDie Zeit (in Millisekunden) wie lange die IP-Adresse blockiert werden soll. 0 kann als permanent benutzt werden.


Rückgabe Werte:

Diese Funktion gibt keinen Wert zurück.


| Pawno Beispiel: 

public OnRconLoginAttempt(ip[], password[], success)
{
    if(!success) // Falls ein falsches Passwort eingegeben wurde.
    {
        BlockIpAddress(ip, 60 * 1000); // blockiert die IP-Adresse für eine Minute.
    }
    return 1;
}
Tipp

Wildcards können für diese Funktion benutzt werden. Als Beispiel für einen Block der IP "6.9.*.*" werden alle IP-Adressen geblockt welche mit den Zahlen 6 und 9 beginnen.
