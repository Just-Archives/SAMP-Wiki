AddStaticPickup

[edit]

AddStaticPickup

Description:

Deze functie kan gebruikt worden als je pickups wilt toevoegen, zoals bijvoorbeeld de rocket launcher. Bijna ieder object uit GTA:SA kan gebruikt worden. Zelfs palm bomen. Wat een pickup moet doen als je hem oppakt ligt aan het type dat je uit hebt gekozen.


Parameters:
(model, type, Float:X, Float:Y, Float:Z)
modelHet model ID van de pickup.
typeHet spawn pickup-type, voor meer informatie kijkt onderaan de pagina.
Float:XDe X-coordinaat voor de pickup positie.
Float:YDe Y-coordinaat voor de pickup positie.
Float:ZDe Z-coordinaat voor de pickup positie.


Voorbeeld: 

public OnGameModeInit( )
{
    // Maak een pickup voor armor
    AddStaticPickup(1242, 2, 1503.3359, 1432.3585, 10.1191);
 
    // Pickup voor health vlakbij de armor
    AddStaticPickup(1240, 2, 1506.3359, 1432.3585, 10.1191);
 
    return 1;
}
[edit]

Extra Functies

De volgende functies zijn mogelijk ook bruikbaar, aangezien ze op een manier erbij horen.

  • CreatePickup: Creer een pickup die daarna ook weer vernietigd kan worden.
  • DestroyPickup: Vernietig een pickup die gemaakt is door CreatePickup.
