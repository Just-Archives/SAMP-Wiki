Page titles

I think page titles need to come back. Some page titles don't display correctly, for example functions which start with lowercase letters (e.g. fopen), underscores don't show (e.g. NetStats_ functions) and so on.

The titles on pages (Template:Title) were originally removed because it seemed silly to have two titles on a page. That's fine for most pages, but these exceptions present a problem. Instead of just having a second title on a page, it may be an idea to use a nice-looking template with some nice css. Unfortunately I am a real novice with HTML/CSS stuff, but I managed to create this:

Function SetPlayerPos
Description Description
SA-MP Versions All


Callback OnPlayerSpawn
Description Description
SA-MP Versions All

Template:Title2

I want some opinions on this.

In fact, it might be nice to bring the description in to this also, so one box for the type (function/callback), name, and then the description under. It'd be a lot of work to change every page, but it can be a gradual change done as pages are edited for other reasons. It'd make the pages look nicer. There could also be a section in the box that shows which version the function was added/removed in.

Another problem is that the templates currently used (Template:Description and Template:Title) can't be used in this. Not sure what could be used instead.
