¡Este/a callback fue agregado/a en SA-MP 0.3a y no funcionará en versiones anteriores!


Description:

Este callback es llamado cuando alguien intenta loguearse como RCON (exitosamente o no).


Parámetros:
(ip[], password[], success)
ip[]ID del jugador que intenta loguearse.
password[]contraseña ingresada por el jugador.
success0 si la contraseña no fue correcta, o en caso contrario 1.


Este callback no maneja returns


Ejemplo: 

public OnRconLoginAttempt(ip[], password[], success)
{
    if(!success)//intento fallido.
    {
        printf("Server: Intento de logueo rcon desde la ip: %s usando el password: %s", ip, password);
        new pIP[16];
        for(new i; i<GetMaxPlayers(); i++)
        {
            GetPlayerIp(i, pIP, 16);
            if(!strcmp(ip, pIP, true))//encontramos el jugador que envio el comando
            {
                SendClientMessage(i, 0xFFFFFFFF, "No debiste intentar loguearte como rcon. Adios");
                Ban(i);
            }
        }
    }
    return 1;
}


Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • IsPlayerAdmin Retorna si un jugador es o no admin RCON.
  • SendRconCommand Envia un comando rcon desde el script.
