Description:

Lie un véhicule dans un intérieur (Utile pour les téléporter dans le 8 Ball et dans le Stade de Derby)


Paramètres:
(vehicleid, interiorid)
vehicleidL'ID du véhicule (pas du modèle).
interioridL'ID de l'intérieur où le véhicule sera lié.


RetourneCette fonction ne retourne pas de valeur spécifique.


Clickez ici pour un guide d'utilisation. 

public OnGameModeInit()
{
    //Autre code
    new vehicle1 = AddStaticVehicle(559,2543.7505,-21.8345,27.1899,52.6054,53,79);
    LinkVehicleToInterior(vehicle1, 6);
    //Autre Code
}
[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • SetVehicleVirtualWorld: Déplace un véhicule dans un monde virtuel.
  • SetPlayerInterior: Déplace le joueur dans un autre intérieur.
