¡Este/a función fue agregado/a en SA-MP 0.3a y no funcionará en versiones anteriores!


Descripción:

Atrae un 3D text label a un jugador.


Parámetros:
(Text3D:id, playerid, Float:OffsetX, Float:OffsetY, Float:OffsetZ)
Text3D:idEl ID del 3d text label que se va a atraer.
playeridLa ID del jugador al que se le atraera el 3D text label.
OffsetXEl desplazamiento X desde el jugador.
OffsetYEl desplazamiento Y desde el jugador.
OffsetZEl desplazamiento Z desde el jugador.


RetornoEsta función no retorna un valor específico


public OnPlayerConnect(playerid)
{
    new Text3D:label = Create3DTextLabel("3D Text Label", 0x008080FF, 30.0, 40.0, 50.0, 40.0, 0);
    Attach3DTextLabelToPlayer(label, playerid, 0.0, 0.0, 0.7);
    return 1;
}

Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • CreatePlayer3DTextLabel Crea un nuevo 3D text label a un jugador.
  • DeletePlayer3DTextLabel Elimina un 3D text label existente de un jugador.
  • UpdatePlayer3DTextLabelText Actualiza un 3d text label de un jugador.


  • Create3DTextLabel Crea un nuevo 3D text label.
  • Delete3DTextLabel Elimina un 3D text label.
  • Attach3DTextLabelToVehicle Atrae un 3D text label a un vehiculo.
  • Update3DTextLabelText Actualiza un 3D text label.
