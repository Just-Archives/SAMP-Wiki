Description:

Du kannst diese Funktion verwenden, um Fahrzeuge in deinen Gamemode einzufügen. Diese Funktion kann nur in den OnGameModeInit und OnFilterScriptInit Callbacks verwendet werden. Sieh dir CreateVehicle an, wenn du während der Spielzeit Fahrzeuge erstellen willst.


Parameter:
(modelid, Float:spawn_x, Float:spawn_y, Float:spawn_z, Float:angle, color1, color2)
modelidDas Fahrzeug Modell, das verwendet werden soll.
Float:spawn_XDie X Koordinate der Position, an der das Fahrzeug erscheinen soll.
Float:spawn_YDie Y Koordinate der Position, an der das Fahrzeug erscheinen soll.
Float:spawn_ZDie Z Koordinate der Position, an der das Fahrzeug erscheinen soll.
Float:angleDie Richtung, in die das Fahrzeug zeigt (Winkel).
color1Die erste Farbe.
color2Die zweite Farbe.


RückgabeDie ID des Fahrzeugs, das erstellt wurde.


public OnGameModeInit()
{
    // Füge eine Hydra ein
    AddStaticVehicle(520, 2109.1763, 1503.0453, 32.2887, 82.2873, 0, 1);
 
    return 1;
}

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • AddStaticVehicleEx: Erstellt ein statisches Fahrzeug mit einer benutzerdefinierten Respawn-Zeit.
  • CreateVehicle: Erstellt ein Fahrzeug.
  • DestroyVehicle: Zerstört (löscht) ein Fahrzeug.
