Description:

Aggiunge un veicolo nella gamemode. Questa funzione può essere usata soltanto nei callback OnGameModeInit e OnFilterScriptInit.
Consulta la funzione Createvehicle se vuoi aggiungere veicoli anche durante il gioco.


Parameters:
(modelid, Float:spawn_x, Float:spawn_y, Float:spawn_z, Float:angle, color1, color2)
modelidID del modello del veicolo.
Float:spawn_XCoordinata X del veicolo.
Float:spawn_YCoordinata Y del veicolo.
Float:spawn_ZCoordinata Z del veicolo.
Float:angleAngolazione del veicolo.
color1ID del colore primario.
color2ID del colore secondario.


Return   Questa funzione non restituisce un valore specifico 

public OnGameModeInit()
{
    // Aggiunge un'Hydra nella gamemode
    AddStaticVehicle(520, 2109.1763, 1503.0453, 32.2887, 82.2873, 0, 1);
 
    return 1;
}

Funzioni simili

  • AddStaticVehicleEx: Aggiunge un veicolo statico con respawn personalizzato
  • CreateVehicle: Crea un veicolo
  • DestroyVehicle: Elimina un veicolo
