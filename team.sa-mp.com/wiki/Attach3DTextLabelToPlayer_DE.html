Diese Funktion/Dieses Callback wurde erstmals in SA:MP 0.3 hinzugefügt und funktioniert nicht in älteren Versionen!


Description:

Befestigt einen 3D Text an einem Spieler.


Parameter:
(Text3D:id, playerid, Float:OffsetX, Float:OffsetY, Float:OffsetZ)
Text3D:idDie ID vom 3D Text, der befestigt werden soll.
playeridDie ID vom Spieler, an dem der Text befestigt werden soll.
OffsetXDer X-Abstand, vom Spieler ausgehend (Die Mitte des Spielers ist 0).
OffsetYDer Y-Abstand, vom Spieler ausgehend (Die Mitte des Spielers ist 0).
OffsetZDer Z-Abstand, vom Spieler ausgehend (Die Mitte des Spielers ist 0).


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


public OnPlayerConnect(playerid)
{
    new Text3D:label = Create3DTextLabel("Hello, I am new here!", 0x008080FF, 30.0, 40.0, 50.0, 40.0, 0);
    Attach3DTextLabelToPlayer(label, playerid, 0.0, 0.0, 0.7);
    return 1;
}

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • CreatePlayer3DTextLabel: Erstellt einen 3D Text für einen Spieler.
  • DeletePlayer3DTextLabel: Löscht einen Spieler-3D-Text.
  • UpdatePlayer3DTextLabelText: Ändert den Text eines Spieler-3D-Textes.


  • Create3DTextLabel: Erstellt einen 3D Text.
  • Delete3DTextLabel: Löscht ein erstelltes 3DTextLabel.
  • Attach3DTextLabelToVehicle: Befestigt einen 3D Text an einem Fahrzeug.
  • Update3DTextLabelText: Ändert den Text eines 3D Textes.
