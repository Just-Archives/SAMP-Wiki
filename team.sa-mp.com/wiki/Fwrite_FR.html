Description:

Écrit dans un fichier.


Paramètres:
(File:handle, string[])
handleLe fichier (retourné via fopen) dans lequel on va écrire.
stringLa chaîne(string) à écrire.


RetourneCette fonction ne retourne pas de valeur spécifique.


public OnGameModeInit()
{
    new File:log = fopen("/ServerLog.txt", io_write); // Ouverture du fichier
    if(log)
    {
        fwrite(log, "Serveur lancé!.\r\n"); // Écrit dans le fichier (\r\n est pour une nouvelle ligne)
        fclose(log); // Fermeture du fichier
    }
    return 1;
}
[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • fread: Lit un fichier.
  • fopen: Ouvre un fichier
  • fclose: Ferme un fichier
  • fremove: Supprime un fichier.
