¡Este/a NPC callback fue agregado/a en SA-MP 0.3a y no funcionará en versiones anteriores!


Description:

Este callback es llamado cuando un NPC sale a un vehiculo.


Nota

Para la version de un jugador (no NPC) mirar OnPlayerEnterVehicle.


Este callback no tiene parametros


Este callback no maneja returns


Ejemplo: 

public OnNPCExitVehicle(vehicleid, seatid)
{
    print("Un NPC abandono su vehiculo.");
    return 1;
}

Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • NPC:OnNPCEnterVehicle Llamado cuando un NPC entra a un vehiculo.
