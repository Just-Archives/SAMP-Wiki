This function was added in SA-MP 0.3a and will not work in earlier versions!


Description:

Mostra una sincronizzazione di una lista di dialoghi ad un giocatore.


Parameters:
(playerid, dialogid, style, caption[], info[], button1[], button2[])
playeridL'ID del giocatore per fargli vedere il dialogo.
dialogidUn ID che assegna questo dialogo a qualcuno, quindi le risposte possono essere eseguiti. Il massimo del dialogid è di 32767.Utilizzando valori negativi per chiudere qualsiasi dialogo aperto.
styleLo stile del dialogo.
caption[]Il titolo nella parte superiore della finestra di dialogo.
info[]Il testo che comprarirà nel dialogo. Usa \n per iniziare una nuova linea \t per tabulare.
button1[]Il tasto nella parte sinistra.
button2[]Il tasto nella parte destra.


Return Values:

This function does not return any specific values.


//esempio per DIALOG_STYLE_INPUT
ShowPlayerDialog(playerid,1,DIALOG_STYLE_INPUT,"Login","Inserisci la tua password qui sotto:","Login","Annulla");
 
//esempio per DIALOG_STYLE_LIST
ShowPlayerDialog(playerid,1,DIALOG_STYLE_LIST,"Il mio Dialogo","1. Linea\r\n2. Linea\r\n3. Linea","Opzione 1", "Opzione 2");

Funzioni correlate

Le seguenti funzioni potrebbe essere utili, dato che sono collegate a tale funzioni in un modo o nell'altro.

  • TextDrawShowForPlayer: Show a textdraw for a certain player.


Related Callbacks

The following callbacks might be useful, as they're related to this function in one way or another.

  • OnDialogResponse: Called when a player responds to a dialog.
