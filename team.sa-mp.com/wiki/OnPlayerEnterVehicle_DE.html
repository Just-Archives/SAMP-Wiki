Description:

Dieses Callback wird aufgerufen, wenn ein Spieler beginnt ein Fahrzeug zu betreten. Der Spieler ist also noch nicht im Fahrzeug, wenn OnPlayerEnterVehicle aufgerufen wird.


Parameter:
(playerid, vehicleid, ispassenger)
playeridDie ID des Spielers.
vehicleidDie ID des Fahrzeugs
ispassenger0 wenn Spieler als Fahrer einsteigt, 1 wenn als Beifahrer.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.



Beispiel

Dieses Beispiel zeigt die ID des Fahrzeugs an, das man gerade betritt. 

public OnPlayerEnterVehicle(playerid, vehicleid, ispassenger)
{
    new string[128];
    format(string, sizeof(string), "Du betritts Fahrzeug %i",vehicleid);
    SendClientMessage(playerid, 0xFFFFFFFF, string);
    return 1;
}

Hinweis

Für ein Callback, das erst aufgerufen wird, wenn der Spieler tatsächlich im Auto sitzt, und nicht nur auf dem Weg dahin ist, siehe OnPlayerStateChange.


Verwandte Callbacks

Die folgenden Callbacks könnten nützlich sein, da sie auf eine Weise mit diesem Callback zu tun haben.

  • OnPlayerExitVehicle: Wird aufgerufen, wenn ein Spieler ein Fahrzeug verlässt.
  • OnPlayerStateChange: Wird aufgerufen, wenn sich der Status eines Spielers ändert.
