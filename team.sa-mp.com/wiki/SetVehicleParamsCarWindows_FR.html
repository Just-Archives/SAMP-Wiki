Ne prenez pas en compte les FR dans les noms

Cette function a été ajoutée dans SA-MP 0.3.7 et ne fonctionnera pas sur des versions antérieures!


Description:

Permet d'ouvrir et de fermer les fenêtres d'un véhicule.


Paramètres:
(vehicleid, driver, passenger, backleft, backright)
vehicleidID du véhicule
driverEtat de la fenêtre conducteur. 1 pour ouvrir, 0 pour fermer.
passengerEtat de la fenêtre passager avant. 1 pour ouvrir, 0 pour fermer.
backleftEtat de la fenêtre arrière gauche (si disponible). 1 pour ouvrir, 0 pour fermer.
backrightEtat de la fenêtre arrière droite (si disponible). 1 pour ouvrir, 0 pour fermer.


[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • SetVehicleParamsCarDoors: Open and close the doors of a vehicle.
  • GetVehicleParamsCarDoors: Retrive the current state of a vehicle's doors.
  • GetVehicleParamsCarWindows: Retrive the current state of a vehicle's windows
