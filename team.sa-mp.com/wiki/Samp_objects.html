Description:

This page contains a list of the custom objects added to SA:MP in the 0.3c, 0.3d, 0.3e, 0.3x and 0.3.7 versions. The objects were extracted, edited and/or created by Cessil, Kalcor, Matite and Micky between September 2010 and April 2015.


Note

If you wish to see images of all SA-MP objects, visit the GTA San Andreas and SA-MP object model IDs made by Prineside DevTools.


Note

SA-MP and Custom object ID usage:
  • 18631 to 19999 (1369 total) reserved for SAMP.IDE
  • 11682 to 12799 (1118 total) reserved for SAMP.IDE
  • 15065 to 15999 (935 total) reserved for CUSTOM.IDE
