Description:

Appelée lorsqu'un joueur change de classe dans la sélection de classe.


Paramètres:
(playerid, classid)
playeridL'ID du joueur qui change de classe.
classidLa classe que visionne actuellement le joueur.


Retourne:

Retourner 0 va empêcher le joueur de changer de classe.


Note

Les ID de classes sont assignés quand AddPlayerClass est utilisé. 


public OnPlayerRequestClass(playerid,classid)
{
    if(classid == 3 && !IsPlayerAdmin(playerid))
    {
        SendClientMessage(playerid,COLOR_GREEN,"Ce skin est réservé aux admins!");
        return 0;
    }
    return 1;
}
[edit]

Callback Relatives

Les Callbacks ci dessous sont indirectement ou directement liées à cette Callback.

  • OnPlayerRequestSpawn: Appelée lorsqu'un joueur appuie sur le bouton 'spawn' dans la sélection de classe.


[edit]

Fonctions relatives

Les fonctions suivantes peuvent êtres utiles car elles concernent indirectement ou non cette callback.

  • AddPlayerClass: Ajoute une classe.
