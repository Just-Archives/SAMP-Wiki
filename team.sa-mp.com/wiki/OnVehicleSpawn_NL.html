OnVehicleSpawn

Beschrijving:

Deze callback wordt opgeroepen wanneer een voertuig spawnt.


Parameters:
(vehicleid)
vehicleidHet ID van het voertuig dat zojuist is gespawnt.


Retourneert:

Deze callback heeft geen specifieke return waarden.


Voorbeeld: 

public OnVehicleSpawn(vehicleid) {
 
	printf("Voertuig: %i is zojusit gespawnt!", vehicleid);
	return 1;
}
[edit]

Extra Callbacks

De volgende callbacks kunnen handig zijn aangezien ze op een manier erbij horen.

  • OnPlayerSpawn: Opgeroepen wanneer een speler spawnt.
  • OnVehicleDeath: Dit wordt opgeroepen wanneer een voertuig werd vernietigd.
