[edit]

SetGravity

Описание:

Устанавливает гравитацию в игре для всех игроков.


Примечание

Гравитация по умолчанию: 0.008


Параметры:
(Float:gravity)
Float:gravityЗначение гравитации (от -50.0 до 50.0).


Возвращаемые значения:

Эта функция всегда возвращает 1, даже когда ей не удалось выполниться, если указанная гравитация превышает лимиты (меньше -50.0 или больше +50.0).


Пример использования: 

//Поставить гравитацию как на луне
SetGravity(0.005);
[edit]

Связанные функции

Следующие функции могут быть полезны, т.к. они так или иначе связаны с текущей функцией.

  • GetGravity: Get the currently set gravity.
  • SetWeather: Set the global weather.
  • SetWorldTime: Set the global server time.
