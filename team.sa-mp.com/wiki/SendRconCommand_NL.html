SendRconCommand

Description:

Bij gebruik van deze functie kun je een RCON command verzenden


Parameters:
(command[])
command[] Command die moet worden verstuurd


SendRconCommand("gmx");
// Dit is een gescripte versie van "/rcon gmx" in-game
// Gmx restart the server
