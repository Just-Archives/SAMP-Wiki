Remote Console IT

La Remote Console o Server Browser è un prompt di comando dove è possibile utilizzare i comandi RCON senza dover essere in gioco o sul server. Utilizzo del Server browser è semplice.

1. Fare click sul server che desidera utilizzare questa funzione.
2. Fare clic sul pulsante nero che assomiglia a un prompt dei comandi proprio accanto al pulsante Impostazioni

3. Inserisci la RCON password. Questa è la stessa password configurata nel file server.cfg.

4. Digitare i comandi

Avviso: Non è necessario digitare "/rcon" prima del comando nel server browser e il comando non funzionerà se lo farai. Per esempio, se si vuole riavviare il server è sufficiente digitare "gmx" poi premere INVIO. Questo è tutto da fare. Divertiti
