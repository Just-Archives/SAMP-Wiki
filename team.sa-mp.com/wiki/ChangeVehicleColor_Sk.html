Description:

Táto funkcia vám dovoľuje zmeniť Hlavnú a Sekundárnu farbu vašeho vozidla počas hry. Neexistuje žiadna funkcia na upravenie tretej farby na aute.


Note

Na niektorých vozidlách sa dá zmeniť len hlavná farba, na niektorých vozidlách farba nejde zmeniť vôbec.


Parameters:
(vehicleid, color1, color2)
vehicleidID vozidla, ktorému chcete zmeniť farbu.
color1Nová hlavná farba vozidla.
color2Nová sekundárna farba vozidla.


Return Values:

This function does not return any specific values.


public OnPlayerEnterVehicle(playerid, vehicleid, ispassenger)
{
    // Zmení hlavnú farbu na čiernu a sekundárnu farbu na bielu
    ChangeVehicleColor(vehicleid, 0, 1);
    return 1;
}
