Een werkende server!
  • echo - Dit is een manier om iets te laten zien op de console zodra de server wordt gestart.
  • lanmode - Als je deze aan zet (1), wordt er meer bandbreedte verbruikt, maar is er minder lag en meer synchronisatie.
  • maxplayers - Maximaal aantal spelers op een gegeven moment.
  • port - De poort van de server Portforward. Standaard is 7777
  • hostname - Dit is de naam voor je server.
  • gamemodeXX (N) (t) - Het is mogelijk om welk gamemode te laten lopen die je wilt. De nummers voor meerdere gamemodes lopen op. (N) is de filename van de gamemode (gamemodes/naam.amx), (t) is het aantal keren spelen voordat de volgende gamemode wordt geladen.
    • gamemode0 lvdm 1
    • gamemode1 rivershell 1
    • gamemode2 area51 1
    • gamemode4 sftdm 1
  • weburl - Dit is extra informatie over je server, in dit geval de website.
  • rcon_password - Dit is een wachtwoord voor RCON, deze moet anders zijn dan changeme, anders start de server niet.
  • filterscripts (N) - Filterscripts zijn scripts die kunnen worden geladen, gescheiden met een spatie. Voorbeeld: filterscripts script1 script2
  • password (p) - Als je een verplicht wachtwoord wilt voor iemand om de server te joinen password xxx


This server directive was added in SA-MP 0.3a and will not work in earlier versions!


  • rcon 0/1 - Als je 0 gebruikt wordt RCON uitgeschakeld.
  • maxnpc - Het maximum aantal npc's in je server.
  • onfoot_rate - Zendratio van te voet data paketten.
  • incar_rate - Zendratio van auto data paketten.
  • weapon_rate - Zendratio van wapen data paketten.
  • stream_distance - De afstand vanaf waar een speler of auto getoond word. Maximum 500.0 [Bron?].
  • stream_rate - Hoe vaak de server moet checken of iets word gestreamd.
