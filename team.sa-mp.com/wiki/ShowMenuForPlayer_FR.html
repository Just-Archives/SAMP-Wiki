Description:

Montre un menu créé précedemment à un joueur.


Paramètres:
(menuid, playerid)
menuidL'ID du menu à montrer. Retourné par CreateMenu.
playeridL'ID du joueur à qui le menu sera montré.


Retourne:

  • 1: La fonction a bien été exécutée.
  • 0: La fonction n'a pas pu être exécutée. Menu et/ou joueur n'existent pas.


new Menu:JoueurTP;
 
public OnGameModeInit()
{
    JoueurTP = CreateMenu(...);
    return 1;
}
 
public OnPlayerCommandText(playerid, cmdtext[])
{
 
    if(strcmp(cmdtext, "/tele", true) == 0)
    {
        ShowMenuForPlayer(JoueurTP, playerid);
        return 1;
    }
    return 0;
}
[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • AddMenuItem: Ajoute un item dans un menu
  • SetMenuColumnHeader: Modifie le titre d'un choix dans un menu
  • CreateMenu: Crée un menu et retourne son ID.


[edit]

Callbacks Relatives

Les Callbacks suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • OnPlayerSelectedMenuRow: Appelée quand un joueur sélectionne une ligne d'un menu.
  • OnPlayerExitedMenu: Callback appelée lorsqu'un joueur sort d'un menu via "enter".
