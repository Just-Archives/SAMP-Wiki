Cette fonction a été ajoutée dans SA-MP 0.3 et n'est pas utilisable dans les versions antérieures!


Description:

Est appelé lorsqu'un véhicule est stream pour un NPC.


Paramètres:
(vehicleid)
vehicleidL'ID du véhicule qui est stream.


Retourne:

Cette callback ne retourne rien, mais doit retourner quelque chose! Regardez ICI pour plus d'informations.


public OnVehicleStreamIn(vehicleid)
{
    SendChat("Super! je viens juste d'apercevoir un véhicule !"); // Ce message s'envoit à chaque fois que le NPC stream un véhicule
    return 1;
}
[edit]

Callback Relatives

Les Callbacks ci dessous sont indirectement ou directement liées à cette Callback.

  • NPC:OnVehicleStreamOut FR: Est appelé lorsqu'un véhicule disparait du champs de stream du NPC
  • NPC:OnPlayerStreamIn FR: Est appelé quand un joueur est stream pour le NPC
  • NPC:OnPlayerStreamOut FR: Est appelé lorsqu'un joueur disparait du champs de stream du NPC
