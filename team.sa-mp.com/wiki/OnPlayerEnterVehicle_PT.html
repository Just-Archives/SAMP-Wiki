OnPlayerEnterVehicle PT

From SA-MP Wiki

Jump to: navigation, search
Pagina Inicial | Basico | Funções | Callbacks



Description:

Esta Callback é chamada quando um jogador começa a entrar em um veiculo, lembrando que o jogador ainda não esta no veiculo no momento em que esta callback é chamada.


Parâmetros:
(playerid, vehicleid, ispassenger)
playeridID do jogador que tenta entrar em um veículo.
vehicleidID do veiculo no qual o jogador esta tentando entrar.
ispassenger0 se entra como motorista. 1 se entra como passageiro.


RetornoEsta função não retorna um valor especifico.


Aqui temos um exemplo que envia uma mensagem com o ID do veiculo que o playerid esta tentando entrar .

Exemplo de uso: 

public OnPlayerEnterVehicle(playerid, vehicleid, ispassenger)
{
    new string[128];
    format(string, sizeof(string), "Você esta entrando no veiculo: %i",vehicleid);
    SendClientMessage(playerid, 0xFFFFFFFF, string);
    return 1;
}

Notas

  • Esta Callback é chamada quando o jogador COMEÇA a entrar no veiculo, não quando ele já esta nele. Para isto use OnPlayerStateChange.
  • Esta Callback ainda e chamada se a entrada for negada para o jogador (ex: o veiculo esta trancado ou cheio).


Callbacks Relacionadas

As seguintes Callbacks também podem ser úteis, pois estão relacionadas a esta Callback.

  • OnPlayerExitVehicle: Chamado quando um jogador deixa o veiculo.
  • OnPlayerStateChange: Chamado quando um jogador muda de estado.


Funções Relacionadas

As seguintes funções podem ser úteis, pois estão relacionadas a esta callback.

  • PutPlayerInVehicle: Coloca um jogador em um veiculo.
  • GetPlayerVehicleSeat: Pega o assento em que o jogador esta.
