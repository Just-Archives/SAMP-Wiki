Description:

Dieses Callback wird aufgerufen, wenn ein Spieler damit beginnt, ein Fahrzeug zu verlassen.


Parameter:
(playerid, vehicleid)
playeridDie ID des Spielers, der das Fahrzeug verlässt.
vehicleidDie ID des Fahrzeugs, das der Spieler verlässt.


Dieses Callback benutzt den Return-Wert nicht.



Wichtiger
Hinweis

Es wird nicht aufgerufen, wenn ein Spieler von einem Motorrad oder Fahrrad fällt, und auch nicht, wenn der Spieler durch Scriptfunktionen wie SetPlayerPos aus dem Fahrzeug gezwungen wird.

Um auch solche Fälle zu erkennen kann OnPlayerStateChange benutzt werden. 


public OnPlayerExitVehicle(playerid, vehicleid)
{
    new string[35];
    format(string, sizeof(string), "INFO: Du verlässt Fahrzeug %i", vehicleid);
    SendClientMessage(playerid, 0xFFFFFFFF, string);
    return 1;
}
[edit]

Verwandte Callbacks

Die folgenden Callbacks könnten nützlich sein, da sie auf eine Weise mit diesem Callback zu tun haben.

  • OnPlayerEnterVehicle: Wird aufgerufen, wenn ein Spieler beginnt ein Fahrzeug zu betreten.
  • OnPlayerStateChange: Wird aufgerufen, wenn sich der Status eines Spielers ändert.
