Requisiti

  • Accesso a Internet (per 100 giocatori in un server si dovvrebbe avere 4Mbps di banda libera in Upload)
  • Una porta UDP forwarded (7777 predefinita) | Guida
  • Un'ip statico sarebbe consigliato, ma non è neccessario.
[edit]

Configurare il server

[edit]

Ottenere i file

Il primo passaggio per aprire il vostro server samp è scaricare Windows Server file da Pagina di download SA-MP trovarai l'ultima versione aggiornata di Windows Server i file sotto Windows Server fai - click su uno dei download, assicuratevi di mettere il file da qualche parte dove si può facilmente ricordare.

[edit]

Configurare il vostro Server

Una volta scaricati i file, estrare (utilizzando il programma che può estrarre i file .rar) i file (non è necessario vedere dove mettete la cartella, come il server non ha bisogno di vedere se si dispone di GTA:SA e SA-MP installato per l'esecuzione). Ora, procedere alla modifica del file server.cfg.

Trova server.cfg e aprila dalla cartella principale con Blocco note o altro editor di testo. Questo file dovrebbe apparire in inglese. Questo file è la configurazione del server, qui si può specificare se lanmode è on o off, i max players, le server port, host name, game modes, filterscripts*, weburl e la tua rcon password.

[edit]

Esecuzione del server

Una volta configurato il server può essere avvito. Per avviralo è sufficiente fare semplicemente click su samp-server.exe e il vostro server sarà in esecuzione.
