Cette fonction a été ajoutée dans SA-MP 0.3z et ne fonctionnera pas dans les versions antérieures!


Description:

Cette fonction donne des informations sur le dernier coup de feu tiré par un joueur.


Paramètres:
(playerid, &Float:fOriginX, &Float:fOriginY, &Float:fOriginZ, &Float:fHitPosX, &Float:fHitPosY, &Float:fHitPosZ)
playeridL'ID du joueur dont on veut récupérer les informations du dernier coup de feu.
&Float:fOriginXLa variable float où stocker les coordonnées X de l'origine du tir.
&Float:fOriginYLa variable float où stocker les coordonnées Y de l'origine du tir.
&Float:fOriginZLa variable float où stocker les coordonnées Z de l'origine du tir.
&Float:fHitPosXLa variable float où stocker les coordonnées X de la destination du tir.
&Float:fHitPosYLa variable float où stocker les coordonnées Y de la destination du tir.
&Float:fHitPosYLa variable float où stocker les coordonnées Z de la destination du tir.


Retourne:

La fonction en elle-même ne retourne pas de valeur spécifique. Les positions sont stockées dans des variables.


public OnPlayerCommandText(playerid, cmdtext[])
{
    if(strcmp(cmdtext, "/derniertir", true) == 0)
    {
        new lsString[128],
            Float:fOriginX, Float:fOriginY, Float:fOriginZ,
            Float:fHitPosX, Float:fHitPosY, Float:fHitPosZ;
 
        GetPlayerLastShotVectors(playerid, fOriginX, fOriginY, fOriginZ, fHitPosX, fHitPosY, fHitPosZ}
        format(lsString, 128, "Dernières informations de tir: Origine: %f, %f, %f. Position touchée: %f, %f, %f", fOriginX, fOriginY, fOriginZ, fHitPosX, fHitPosY, fHitPosZ);
        SendClientMessage(playerid, -1, lsString);
        return 1;
    }
    return 0;
}
[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • GetPlayerWeaponData: Récupère des informations à propos des armes que le joueur a.
  • GetPlayerWeapon: Permet de savoir quelle arme le joueur porte actuellement.
