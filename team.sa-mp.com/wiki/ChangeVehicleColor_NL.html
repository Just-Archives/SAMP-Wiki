ChangeVehicleColor

Description:

Bij gebruik van deze functie kan je de kleuren van je voertuigen veranderen.


Parameters:
(voertuigid, kleur1, kleur2)
voertuigidHet ID van het voertuig, dus niet het modelid.
kleur1De primaire kleur voor het voertuig.
kleur2De secondaire kleur voor het voertuig.


public OnPlayerEnterVehicle(playerid, vehicleid, ispassenger)
 {
     // Verander de primaire kleur naar wit en de secondaire naar zwart.
     ChangeVehicleColor(vehicleid, 0, 1);
 
     return 1;
 }
[edit]

Extra Functies

De volgende functies zijn mogelijk ook bruikbaar, aangezien ze op een manier erbij horen.

  • AddVehicleComponent: Voeg een onderdeel toe aan je voertuig, bijvoorbeeld nitro.
  • CreateVehicle: Maak een nieuw voertuig.
  • ChangeVehiclePaintjob: Geef een paintjob aan een bepaald voertuig.
