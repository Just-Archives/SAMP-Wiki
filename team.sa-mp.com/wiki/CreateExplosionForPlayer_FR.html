Cette callback a été ajoutée dans SA-MP 0.3z et ne fonctionnera pas dans les versions antérieures!


Description:

Crée une explosion uniquement pour un seul joueur.


Paramètres:
(playerid, Float:X, Float:Y, Float:Z, type, Float:Radius)
playeridL'ID du joueur qui verra l'explosion.
Float:XLes coordonnées X où créer l'explosion.
Float:YLes coordonnées Y où créer l'explosion.
Float:ZLes coordonnées Z où créer l'explosion.
typeLe type d'explosion.
Float:RadiusLe rayon de l'explosion.


RetourneCette fonction ne retourne pas de valeur spécifique.


if (strcmp(cmdtext, "/brulemoi", true) == 0)
{
    new Float:POS[3];
    GetPlayerPos(playerid, POS[0], POS[1], POS[2]);
    CreateExplosionForPlayer(playerid, POS[0], POS[1], POS[2], 1, 10.0);
    return 1;
}
[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • CreateExplosion: Crée une explosion visible pour tous les joueurs.
