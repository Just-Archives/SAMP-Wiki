LAN IT

Jump to: navigation, search

Giocare in LAN a SA-MP è semplice, purché la vostra LAN è già impostata.

  • Passo 1: Scegliere un computer che avvia il server, allora fai (Guarda Windows Server per maggior informazioni).
  • Passo 2: Per Ottenere l'IP del computer che tiene il server andate su Pannello di controllo -> Configurazione Connessione -> Local Area Connection (Click Destro -> Proprietà), poi vai giù fino a "Internet Protocol (TCP/IP)" e clicca proprietà.
  • Passo 3: Avviate il Server!
  • Passo 4: Collegare tutti al server aggiungendo l'IP del server ai tuoi preferiti in SA: MP (l'IP è quello dal punto 2)
  • Passo 5: Giocate!

Avviso Importante: Se non vedi il server nel browser, cerca di aggiungere 127.0.0.1:port, nella lista favorites ma questo problema è raro.
