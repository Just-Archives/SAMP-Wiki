SpecialActions

Это ID'ы специальных действий, которые могут быть установлены в функции SetPlayerSpecialAction или возвращены в функции GetPlayerSpecialAction. 

0  - SPECIAL_ACTION_NONE              // Пустое действие (остановить анимацию)
2  - SPECIAL_ACTION_USEJETPACK        // Использовать джетпак
5  - SPECIAL_ACTION_DANCE1            // Танец 1
6  - SPECIAL_ACTION_DANCE2            // Танец 2
7  - SPECIAL_ACTION_DANCE3            // Танец 3
8  - SPECIAL_ACTION_DANCE4            // Танец 4 (Стриптиз)
10 - SPECIAL_ACTION_HANDSUP           // Руки вверх
11 - SPECIAL_ACTION_USECELLPHONE      // Использование телефона
12 - SPECIAL_ACTION_SITTING           // Присесть
13 - SPECIAL_ACTION_STOPUSECELLPHONE  // Кладёт телефон обратно

Эти специальные действия появились в SA-MP 0.3 и не работает в более ранних версиях! 


1  - SPECIAL_ACTION_DUCK              // Наклон (не может быть установлено)
3  - SPECIAL_ACTION_ENTER_VEHICLE     // Анимация посадки в транспорт (не может быть установлено)
4  - SPECIAL_ACTION_EXIT_VEHICLE      // Анимация выхода из транспорта (не может быть установлено)
20 - SPECIAL_ACTION_DRINK_BEER        // Пьёт пиво (повышается уровень опьянения)
21 - SPECIAL_ACTION_SMOKE_CIGGY       // Курит сигару
22 - SPECIAL_ACTION_DRINK_WINE        // Пьёт вино
23 - SPECIAL_ACTION_DRINK_SPRUNK      // Пьёт Sprunk из алюминиевой банки
68 - SPECIAL_ACTION_PISSING           // Справляет малую нужду (нет в a_samp.inc)

Это специальное действие появилось в SA-MP 0.3e и не работает в более ранних версиях! 


24  - SPECIAL_ACTION_CUFFED              // Руки за спиной (в наручниках)

Это специальное действие появилось в SA-MP 0.3x и не работает в более ранних версиях! 


 25 - SPECIAL_ACTION_CARRY               // Анимация "грузчика", делает невозможным спринт, прыжки и удары
