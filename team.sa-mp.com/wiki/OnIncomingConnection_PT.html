Esta callback foi adicionada no SA-MP 0.3z R2-2 e não irá funcionar em versões anteriores!


Descrição:

Esta callback informa o servidor que um ip está conectando. Você pode bloquear a conexão usando a função BlockIpAddress.


Parâmetros:
(playerid, ip_address[], port)
playeridID do jogador
ip_address[]O IP do jogador
portA porta utilizada pelo cliente


Valores de retorno:

Esta callback não lida com retornos.


Exemplo de uso: 

public OnIncomingConnection(playerid, ip_address[], port)
{
    printf("ID %d está conectando [IP: %s:%i]", playerid, ip_address, port);
    return 1;
}
[edit]

Callbacks Relacionadas

As seguintes Callbacks também podem ser úteis, pois estão relacionadas a esta Callback.

  • OnPlayerConnect: Chamado quando um jogador conecta ao servidor.
  • OnPlayerDisconnect: Chamado quando um jogador desconecta do servidor.


[edit]

Funções Relacionadas

As seguintes funções podem ser úteis, pois estão de alguma forma, relacionadas a esta função.

  • BlockIpAddress: Bloqueia um endereço de IP por um tempo determinado.
  • UnBlockIpAddress: Desbloqueia um IP que foi bloqueado anteriormente.
