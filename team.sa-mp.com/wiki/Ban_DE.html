Description:

Diese Funktion kann verwendet werden, um einen Spieler, der gerade auf dem Server ist, vom Spiel auszuschließen (zu "bannen"). Dieser Ban ist IP-basiert und wird in "samp.ban" im Hauptverzeichnis des Servers abgespeichert. BanEx erlaubt, einen Grund anzugeben. Außer diese Funkionenen zu verwenden kann man auch mit den RCON-Funktionen "banip" und "unbanip" ent-/bannen.


Parameter:
(playerid)
playeridDie ID des Spielers, der gebannt werden soll.
RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


public OnPlayerCommandText(playerid, cmdtext[])
{
    if(strcmp(cmdtext, "/banme", true) == 0)
    {
        // Bannt den Spieler, der diesen Befehl verwendet.
        Ban(playerid);
        return 1;
    }
}

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • BanEx: Bannt einen Spieler mit dem angegeben Grund.
  • Kick: Kickt einen Spieler vom Server.
