Objekte, Spieler, Fahrzeuge und alles ähnliche verwendet IDs. Einige fangen bei 0 an zu zählen, andere bei 1.

Typ Start-ID
3D Text 0
Datei 0
GangZone 0
Objekt 1
Pickup 0
Spieler 0
Spielerklasse 0
TextDraw / Spielerseitiges TextDraw 0
Fahrzeug 1

Wie man sieht, fangen Fahrzeuge und Objekte bei 1 an, nicht bei 0! Wenn man also plant, diese in einem Array mit der ID abzuspeichern, muss man die ID aus CreateObject/CreateVehicle/... verwenden und beim manuellen durchsuchen der Arrays 1 vom Index abziehen, um das richtige Ergebnis zu bekommen.
