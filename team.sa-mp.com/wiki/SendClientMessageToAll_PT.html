SendClientMessageToAll

Descrição:

Mostra uma mensagem no chat de todos os jogadores. Isto é o mesmo que um loop de SendClientMessage_PT.


Parâmetros:
(color, const message[])
colorA cor da mensagem(0xRRGGBBAA formato Hexadecimal).
const message[]A mensagem a ser enviada (máximo 144 caracteres).


Valores de retorno:

Esta função retorna sempre 1 (verdadeiro/true).



Notas
Importantes

Evite utilizar especificadores de formatação na mensagem sem antes formatar a string a enviar. Caso contrário isto irá resultar em um server crash.


Exemplo de uso: 

public OnPlayerCommandText(playerid, cmdtext[])
{
    if (strcmp(cmdtext, "/hello", true) == 0)
    {
        // Envia uma mensagem para todos os jogadores
        SendClientMessageToAll(-1, "Olá!");
        return 1;
    }
    return 0;
}
[edit]

Funções Relacionadas

As seguintes funções podem ser úteis, pois estão de alguma forma, relacionadas a esta função.

  • SendClientMessage: Envia uma mensagem para um determinado jogador.
  • SendPlayerMessageToAll: Force a player to send text for all players.
