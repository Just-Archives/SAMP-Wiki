AddStaticPickup ES

From SA-MP Wiki

Jump to: navigation, search
Inicio | Funciones | Callbacks | Scripting Basics | Server Plugins | Tutoriales



Note

Esta función no retorna la ID del pickup creado. Si deseas crear un pickup para usar en, por ejemplo, OnPlayerPickUpPickup, usa CreatePickup, ya que este si retorna la ID del pickup.


Descripción:

Esta función agrega un pickup "estático". Estos pickups pueden ser armas, armadura o vida, etc., y pueden ser agregados sin tener que programar la función que cumplirán (Por ejemplo, si agregas un pickup de arma, al recoger el pickup, se le dará el arma automáticamente, sin necesidad de usar OnPlayerPickUpPickup).


Parámetros:
(model, type, Float:X, Float:Y, Float:Z, virtualworld)
modelID del modelo del pickup.
typeEl tipo de pickup.
Float:XLa coordenada X donde se creará el pickup.
Float:YLa coordenada Y donde se creará el pickup.
Float:ZLa coordenada Z donde se creará el pickup.
virtualworldID del mundo virtual en el que se creara el pickup. Usa -1 para crearlo en todos los mundos virtuales.


Retornos/Devoluciones:

1 si el pickup fue creado exitosamente.

0 si no se pudo crear.


Ejemplo: 

public OnGameModeInit()
{
    // Crea un pickup de armadura
    AddStaticPickup(1242, 2, 1503.3359, 1432.3585, 10.1191, 0);
 
    // Crea un pickup de vida, justo al lado de la armadura
    AddStaticPickup(1240, 2, 1506.3359, 1432.3585, 10.1191, 0);
 
    return 1;
}

Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • CreatePickup Crea un pickup (retornando la id del mismo).
  • DestroyPickup Destruye un pickup.


Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • OnPlayerPickUpPickup Llamado cuando un jugador toma un pickup.
