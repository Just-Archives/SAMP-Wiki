Description:

Diese Funktion kann verwendet werden, um eine Nachricht an einen bestimmten Spieler in jeder beliebigen Farbe zu senden. Die ganze Zeile in der Chatbox wird in der gewählten Farbe sein, bis Colour-Embedding verwendet wird (0.3c oder später).


Parameter:
(playerid, color, const message[])
playeridDie ID des Spielers, der die Nachricht empfangen soll.
colorDie Farbe, in der die Nachricht erscheinen soll.
const message[]Die Nachricht, die gesendet werden soll. Maximale Zeichenanzahl: 144
RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


public OnPlayerConnect(playerid)
{
    // Sendet "Willkommen auf meinem Server!" an den Spieler playerid in blau (hexcode)
    SendClientMessage(playerid, 0x0000FFFF, "Willkommen auf meinem Server!");
    return 1;
}

Hinweis

Verwende Colour-Embedding für mehrere Textfarben.


[edit]

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • SendClientMessageToAll: Sendet eine Nachricht an alle Spieler.
  • SendPlayerMessageToPlayer: Lässt einen Spieler eine Nachricht an einen anderen Spieler senden.
  • SendPlayerMessageToAll: Lässt einen Spieler eine Nachricht an alle Spieler senden.
