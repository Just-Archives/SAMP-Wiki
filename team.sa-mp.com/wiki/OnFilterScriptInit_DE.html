Description:

Dieses Callback wird aufgerufen, wenn ein Filterscript geladen wird. Es funktioniert lediglich im Filterscript, und nicht im Script des Gamemodes.


Dieses Callback hat keine Parameter.


Dieses Callback benutzt den Return-Wert nicht.


public OnFilterScriptInit()
{
    print("Filterscript geladen.");
    return 1;
}

Verwandte Funktionen

Die folgenden Funktionen könnten interessant sein, da sie etwas mit dem Callback zu tun haben.

  • OnFilterScriptExit: Wird aufgerufen, wenn ein Filterscript beendet wird.
  • OnGameModeInit: Wird aufgerufen, wenn ein Gamemode gestartet wird.
  • OnGameModeExit: Wird aufgerufen, wenn der Gamemode beendet wird.
