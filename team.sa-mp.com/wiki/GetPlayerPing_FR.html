Description:

Récupère le ping(temps de latence) d'un joueur.


Paramètres:
(playerid)
playeridL'ID du joueur dont on va récupérer le ping.


Retourne:

Le ping courant du joueur (le ping est en millisecondes!).


public pingchecktimer(playerid)
{
    // Exclu le joueur si il a un haut ping
    if(GetPlayerPing(playerid) > 1000) Kick(playerid);
    return 1;
}

Note
Importante

Les ping des joueurs peuvent parfois valoir 65535 lors de la connexion du joueur.



[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • GetPlayerIp: Récupère l'IP d'un joueur.
  • GetPlayerName: Récupère le nom (pseudo) d'un joueur.
  • GetPlayerVersion: Récupère la version (de SA:MP) du joueur.
