GetPlayerInterior

Описание:

Позволяет узнать, в каком интерьере находится игрок в данный момент. Список всех известных интерьеров и их координаты можно найти здесь.


Параметры:
(playerid)
playeridID игрока.


Возвращаемые значения:

ID интерьера, в котором находится игрок или 0, если игрок не подключен.


Примечание

Всегда возвращает 0 для NPC.


Пример использования: 

public OnPlayerCommandText(playerid, text[])
{
    if (strcmp(cmdtext, "/int", true) == 0)
    {
        new string[128];
        format(string, sizeof(string), "Interior: %i", GetPlayerInterior(playerid));
        SendClientMessage(playerid, 0xFF8000FF, string);
        return 1;
    }
    return 0;
}
