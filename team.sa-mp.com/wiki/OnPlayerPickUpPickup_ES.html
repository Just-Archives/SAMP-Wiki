OnPlayerPickUpPickup ES

Descripción:

Este callback es llamado cuando un jugador recoge un pickup.


Parámetros:
(playerid, pickupid)
playeridID del jugador que recogio el pickup.
pickupidID del pickup recogido.


Ejemplo: 

new vida;
 
public OnGameModeInit()
{
    vida = CreatePickup(1240, 3, 0.0, 0.0, 0.0);
    return 1;
}
 
public OnPlayerPickUpPickup(playerid, pickupid)
{
    if(pickupid == vida)SetPlayerHealth(playerid, 100.0);
    return 1;
}
[edit]

Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • CreatePickup ES: Crea un nuevo pickup.
  • DestroyPickup ES: Destruye un pickup.
