This function was added in SA-MP 0.3a and will not work in earlier versions!


Description:

Seteaza X, Y si Z ca Viteza



Important
Note

Aceasta functie nu functioneaza pe masinile ne-ocupate sau pe trenuri


Parameters:
(vehicleid, Float:x, Float:y, Float:z)
vehicleidID-ul masinii care va capata viteza
Float:xCantitatea de viteza in directia X.
Float:yCantitatea de viteza in directia Y.
Float:zCantitatea de viteza in directia Z.


Return Values:

This function does not return any specific values.


public OnPlayerCommandText(playerid, cmdtext[])
{
    if (!strcmp("/carjump", cmdtext))
    {
	if(IsPlayerInAnyVehicle(playerid))
        SetVehicleVelocity(GetPlayerVehicleID(playerid), 0.0, 0.0, 0.2);
	return 1;
    }
    if (!strcmp("/speedup", cmdtext))
    {
	if(IsPlayerInAnyVehicle(playerid))
	{
		new Float:X,Float:Y,Float:Z;
		GetVehicleVelocity(GetPlayerVehicleID(playerid), X, Y, Z);
        SetVehicleVelocity(GetPlayerVehicleID(playerid), X * 1.6, Y * 1.6, Z * 1.6);
	}
	return 1;
    }
}
  • SetPlayerVelocity: Set a player's velocity.
  • GetPlayerVelocity: Get a player's velocity.
  • GetVehicleVelocity: Get a vehicle's velocity.
  • SetVehicleAngularVelocity: Set the angular velocity of a vehicle.
