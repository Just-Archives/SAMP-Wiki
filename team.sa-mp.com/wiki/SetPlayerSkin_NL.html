[edit]

SetPlayerSkin

Description:

Met deze functie is het mogelijk om de skin van een speler te wijzigen die eerder al is toegewezen.


Parameters:
(playerid, skin)
playeridHet ID van de speler waarvan je de skin wil veranderen.
skinDe skin die je wilt toewijzen aan een speler.


public OnPlayerDeath(playerid, killerid, reason)
{
    // Controle of het een echte speler is.
    if (killerid != INVALID_PLAYER_ID)
    {
       // Skin van de 'killer' veranderen naar CJ.
       SetPlayerSkin(killerid, 0);
    }
    return 1;
}
[edit]

Extra Functies

De volgende functies zijn mogelijk ook bruikbaar, aangezien ze op een manier erbij horen.

  • AddPlayerClass: Hiermee kan je een class toevoegen om te kiezen.
  • AddPlayerClassEx: Dit is hetzelfde als AddPlayerClass, maar dan met een team parameter.
  • SetSpawnInfo: Bij gebruik van SetSpawnInfo kan je de spawn informatie veranderen voor een speler.
