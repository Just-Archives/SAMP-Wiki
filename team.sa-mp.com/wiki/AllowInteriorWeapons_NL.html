AllowInteriorWeapons

[edit]

AllowInteriorWeapons

Description:

Met deze functie kan je spelers toestaan wapens te gebruiken of juist niet te gebruiken in gebouwen, zoals Ammunation, MaddDoggs Mansion.


Parameters:
(toestaan)
toestaan1 is toestaan, 0 is niet toestaan van gebruik van wapens in gebouwen.


Voorbeeld: 

public OnGameModeInit( )
{
    // Dit zal wapens toestaan in gebouwen.
    AllowInteriorWeapons(1);
    return 1;
}

Extra Functies

De volgende functies zijn mogelijk ook bruikbaar, aangezien ze op een manier erbij horen.

  • SetPlayerInterior: Hiermee kan je het interior wijzigen voor een persoon.
  • SetPlayerVirtualWorld: Met gebruik van deze functie kan je spelers in een interior plaatsen waar niemand anders is.
