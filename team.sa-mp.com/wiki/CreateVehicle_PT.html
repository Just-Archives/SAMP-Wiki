Description:

Cria um veículo durante o jogo. Pode ser usado no lugar de AddStaticVehicle.



Nota
Importante

Trens só podem ser criados usando AddStaticVehicle e AddStaticVehicleEx.


Parâmetros:
(modelid, Float:x, Float:y, Float:z, Float:angle, color1, color2, respawn_delay)
modelidO modelo do veículo.
Float:XA coordenada X para o veículo.
Float:YA coordenada Y para o veículo.
Float:ZA coordenada Z para o veículo.
Float:angleângulo do veículo - Angulo.
color1Cor pimária.
color2Cor secundária.
respawn_delayO tempo em segundos para o veículo dar respawn, quando ele ficar sem motorista. Usando -1, o veículo nunca irá dar respawn.


Valores de retorno:

  • O ID do veículo criado (1 - MAX_VEHICLES).
  • INVALID_VEHICLE_ID (65535) Se o veículo não foi criado (Chegou no limite máximo de veículos ou modelo inválido).


public OnGameModeInit()
{
    // Add um Hydra ao jogo com o tempo de respawn de 60 segundos
    CreateVehicle(520, 2109.1763, 1503.0453, 32.2887, 82.2873, 0, 1, 60);
    return 1;
}
[edit]

Funções Relacionadas

As seguintes funções podem ser úteis, pois estão de alguma forma, relacionadas a esta função.

  • DestroyVehicle: Destrói um veículo.
  • AddStaticVehicle: Add a static vehicle.
  • AddStaticVehicleEx: Add a static vehicle with custom respawn time.


[edit]

Callbacks Relacionadas

As seguintes Callbacks podem ser úteis, pois estão relacionadas de alguma forma a esta função.

  • OnVehicleSpawn: Called when a vehicle respawns.
