[edit]

ispacked

Note: This function name starts with a lowercase letter.


Description:

Checks if the given string is packed.


Parameters:
(const string[])
stringThe string to check.


Return Values:

1 if the string is packed, 0 if it's unpacked.


Example Usage: 

// Create a packed string
new string[24 char];
 
if(ispacked(string))
{
   print("The string is packed.");
}
