Descripción:

Obtener la dirección IP (Internet Protocol) de un jugador especificado y almacenarla en una cadena


Parámetros:
(playerid, ip[], len)
playeridLa ID del jugador a extraer la dirección IP
ip[]La cadena para almacenar la dirección IP del jugador.
lenLa longitud máxima de la dirección IP (16 recomendado).


Return Values:

1 si la función se ejecutó correctamente, 0 si falló.



Notes

  • PAWN distingue mayúsculas y minúsculas. GetPlayerIP no funcionará.
  • Esta función no funcionará si se usa en OnPlayerDisconnect ES porque el jugador ya estará desconectado. Retornará una IP inválida (255.255.255.255). Guarda la IP del jugador en OnPlayerConnect ES si necesitas usarla en OnPlayerDisconnect ES. 


public OnPlayerConnect(playerid)
{
    new ip[16];
    GetPlayerIp(playerid, ip, sizeof(ip));
    if(!strcmp(plrIP, "127.0.0.1")) SendClientMessage(playerid, 0xFFFFFFFF, "Bienvenido a tu servidor, maestro :)");
    return 1;
}
[edit]

Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • NetStats_GetIpPort: Get a player's IP and port.
  • GetPlayerName: Get a player's name.
  • GetPlayerPing: Get the ping of a player.
  • GetPlayerVersion: Get a player's client-version.


[edit]

Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • OnIncomingConnection Ejecutada cuando un jugador intenta conectar con el servidor.
  • OnPlayerConnect Llamado cuando se conecta un jugador.
  • OnPlayerDisconnect Llamado cuando un jugador se desconecta del servidor.
