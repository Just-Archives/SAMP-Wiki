Requerimentos

  • Acesso à Internet (um servidor completo com limite pra 100 jogadores usa aproximadamente 4Mbps)
  • Uma porta UDP aberta (a porta 7777 é a padrão) | Guia
  • É bom ter um IP estático, mas não é necessário
[edit]

Configurando o servidor

[edit]

Obtendo os arquivos

O primeiro passo pra configurar seu servidor de samp é baixar os arquivos do Windows Server da página de downloads do samp. Lá você vai achar a atual versão dos arquivos do Windows Server abaixo da seção Windows Server - clique em um dos links de download pra começar o download, tenha certeza que você colocou o arquivo em algum lugar do qual você consiga se lembrar depois.

[edit]

Configuração do servidor

Uma vez que os arquivos foram baixados, extraia (usando algum programa que consiga extrair arquivos .rar) os arquivos (não precisa extraí-los pra uma pasta especifica, já que o servidor não precisa saber se você tem ou não o GTA:SA e o SA-MP instalado pra ser rodado). Agora, prossiga editando seu arquivo server.cfg.

Ache o arquivo server.cfg na pasta raíz para os arquivos do seu servidor e abra-o com o Bloco de Notas (Notepad) ou outro editor de texto. Esse arquivo deve aparecer pra você em Inglês legivel. Esse arquivo é pra configuração do seu servidor, aqui você pode especificar se o modo lan (lanmode) está ligado ou desligado (on ou off), o maximo de jogadores (max_players), a porta que o servidor usa (server_port), hostname do servidor (hostname), gamemodes, filterscripts*, URL da web (weburl) e sua senha RCON (rcon_password)

[edit]

Rodando o servidor

Uma vez que você configurou seu servidor, você pode rodá-lo. Para rodá-lo, simplesmente abra o arquivo samp-server.exe e seu servidor estará aberto.

[edit]

Mantendo o servidor online

Com opções anti-crash limitadas no Servidor de SA-MP, há uma grande possibilidade de receber Segmentation Faults ou qualquer outro erro por seu gamemode. Isso vai desligar o servidor, e você terá que abri-lo novamente.

Esse programa vai reiniciar o samp-server.exe se o servidor SA-MP travar. Tudo que você precisa fazer é salvar um arquivo de texto com a extensão .bat com o seguinte conteúdo: 

:Start
samp-server.exe
goto start

Coloque o arquivo no diretório onde o samp-server.exe está e inicie-o. Digite exit no console do servidor pra testar se o arquivo está funcionando.
