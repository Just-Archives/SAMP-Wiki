¡Este/a función fue agregado/a en SA-MP 0.3z R2-2 y no funcionará en versiones anteriores!


Descripción:

Bloquear la comunicación entre una dirección IP y el servidor, durante cierto periodo de tiempo (milisegundos) si asi lo desea.


Parámetros:
(ip_address[], timems)
ip_addressIP que se bloqueará.
timemsMilisegundos que se bloqueará la dirección IP.


RetornoEsta función no retorna un valor específico


public OnGameModeInit()
{
    BlockIpAddress("127.0.0.1", 0);
    return 1;
}

Nota

Para bloquear una IP con tiempo indefinido, el parámetro "timems" debe quedar en 0.


Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • OnPlayerConnect Llamado cuando se conecta un jugador.
  • OnPlayerDisconnect Llamado cuando un jugador se desconecta del servidor.
  • NPC:OnNPCConnect Llamado cuando un NPC se conecta al servidor.
  • NPC:OnNPCDisconnect Llamado cuando un NPC se desconecta del servidor.
  • OnIncomingConnection Ejecutada cuando un jugador intenta conectar con el servidor.


Funciones relacionadas

Las siguientes funciones pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • UnBlockIpAddress Permite las conexiones entre una IP y el servidor.
  • ConnectNPC Conecta un NPC al servidor.
