[edit]

Pawno:

Pawno ist der meist verwendestete Editor und wurde von Spooky gemacht! Er ist beim Windows Server mitgeliefert!


Vorteile:

Verbraucht wenig Speicher. Perfekt für kleinere Scripte. Funktions Liste neben dem Script. Mitgeliefert beim Winows Server.

Nachteile:

Bei größeren Scripts schlechte übersicht. Editor ist einfach gestaltet. Du benötigst Adminstrator Rechte auf Vista um ihn zu starten



[edit]

SA-MP Editor

Der zweite Editor ist der SA-MP Editor, dieser ist ziemlich neu. Der SA-MP Editor hat die meisten features von allen. Er ist für größere Scripts sehr gut geeignet und wurde von Peter entwickelt



Vorteile:

Sehr viele Features wie zusammenklappen. IntelliSense® und Tooltips. Projekt Support. Öffne mehere Files in Tabs. Mehr Compile möglichkeiten.


Nachhteile:

Verbraucht mehr Speicher (34mb statt 3mb) Langsam auf älteren PC´s Ist immer noch in Beta. Man benötigt Frameworks 2

Download: Hier klicken!


[edit]

PawnoX

PawnoX ist eine verbesserte Version von Pawno mit vielen neuen Features



Vorteile

Netter, kompakter Editor. Mehr Compil Optionen als bei Pawno. Mehr Buttons in der Main Toolbar.

Nachteile:

Wird nur langsam verbessert! Langsamer als Pawno.
