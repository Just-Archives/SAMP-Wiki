SelectTextDraw

Description:

Activer la selection de TextDraw pour un utilisateur.


Note

Fonctionne uniquement avec les versions supérieurs à la 0.3e


Paramètres:
(playerid, hovercolor)
playeridL'ID du joueur à qui on permet de choisir un TextDraw.
hovercolorLa couleur de la selection.


Note

Vous devez rendre vos TextDraw selectionnable avant d'utiliser cette fonction TextDrawSetSelectable_FR ou PlayerTextDrawSetSelectable_FR.


RetourneCette fonction ne retourne pas de valeur spécifique.


Exemple d'utilisation: 

public OnPlayerCommandText(playerid, cmdtext[])
{
    if(!strcmp(cmdtext, "/tdselect", true))
    {
        SelectTextDraw(playerid, 0x00FF00FF); // La couleur de selection est le vert!
        SendClientMessage(playerid, 0xFFFFFFFF, "SERVER: Selectionnez l'un des TextDraw's");
        return 1;
    }
    return 0;
}
