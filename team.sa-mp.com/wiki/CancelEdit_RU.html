CancelEdit

Описание:

Позволяет выключить режим редактирования объекта для игрока


Данный(-ая) функция был(а) добавлен(a) в SA-MP 0.3e и не будет работать в более ранних версиях!


Параметры:
(playerid)
playeridID игрока, для которого следует остановить редактирование


Возвращаемые значения:

Эта функция ничего не возвращает.


Пример использования: 

public OnPlayerCommandText(playerid, cmdtext[])
{
    if(!strcmp(cmdtext, "/stopedit", true))
    {
        CancelEdit(playerid);
        SendClientMessage(playerid, 0xFFFFFFFF, "SERVER: Вы остановили редактирование объекта!");
        return 1;
    }
    return 0;
}
