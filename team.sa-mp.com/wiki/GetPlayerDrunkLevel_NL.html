GetPlayerDrunkLevel

This function was added in SA-MP 0.3a and will not work in earlier versions!


Beschrijving:

Bekijk de spelers drunklevel. Als het level kleiner is dan 2000, is de speler nuchter. De spelers drunklevel zal automatisch zachtjes dalen (26 levels per seconde). Hoe hoger het drunklevel hoe meer de spelers camera word beinvloed, en ook de driving handling van de auto. Het drunklevel zal stijgen wanneer een speler drunkt van een fles (Je kan SetPlayerSpecialAction gebruiken om ze flessen te geven.).


Parameters:
(playerid)
playeridDe speler waarvan je de drunklevel wil hebben


Retourneert:

Een integer met het drunklevel van een speler.


Voorbeeld: 

public OnPlayerStateChange(playerid, oldstate, newstate)
{
    if(newstate == PLAYER_STATE_DRIVER && GetPlayerDrunkLevel(playerid) > 1999)
    {
        SendClientMessage(playerid,0xFFFFFFFF,"Niet drinken en rijden tegelijk!");
        RemovePlayerFromVehicle(playerid);
    }
}
Tip

Je kan ApplyAnimation gebruiken om een dronken animatie te gebruiken.
