SetPlayerShopName

Diese Funktion wurde erstmals in SA:MP 0.3 hinzugefügt und funktioniert nicht in älteren Versionen!


Beschreibung:

Lädt ein Interior-Script für einen Spieler (zum Beispiel einen Ammunation Verkäufer)


Parameter:
(playerid, shopname[])
playeridDie ID des Spielers, für den das Script geladen werden soll.
shopname[]Das Shop script, das geladen werden soll.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


Hinweis

Casinos werden von dieser Funktion nicht unterstützt.


Beispiel: 

public OnPlayerCommandText(playerid, cmdtext[])
{
    if(!strcmp("/enter", cmdtext))
    {
        SetPlayerInterior(playerid, 5);
        SetPlayerPos(playerid, 372.5565, -131.3607, 1001.4922);
        SetPlayerShopName(playerid,"FDPIZA");
        SendClientMessage(playerid,0xFFFFFFFF,"Willkommen im Pizza Stack!");
        return 1;
    }
    return 0;
}
[edit]

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • DisableInteriorEnterExits: Deaktiviert die gelben Tür-Marker.
  • SetPlayerInterior: Besimmt die Interior ID für einen Spieler.
