OnPlayerClickPlayer

Этот обратный вызов был добавлен в SA-MP 0.3a и не работает в более ранних версиях!


Описание:

Вызывается при нажатии в списке игроков на игрока.


Параметры:
(playerid, clickedplayerid, source)
playerid ID игрока
clickedplayerid ID нажатого игрока
source Тип нажатия


Возвращаемые значения:

  • 1 - Указывает, что этот обратный вызов не будет передан другим filterscript'ам.
  • 0 - Этот обратный вызов будет передан в следующий filterscript.
  • Это всегда вызывается первым из filterscript'ов.


Примечание

Пока существует только один тип 'source' (0 - CLICK_SOURCE_SCOREBOARD). Существование этого аргумента говорит о том, что в будущем могут быть добавлены и другие типы.


Пример использования: 

public OnPlayerClickPlayer(playerid, clickedplayerid, source)
{	
	new message[50];
        new nickname[MAX_PLAYER_NAME]; 
        GetPlayerName(clickedplayerid,nickname,sizeof(nickname));
	format(message, sizeof(message), "Вы нажали на %s(ID:%d)", nickname, clickedplayerid);
	SendClientMessage(playerid, 0xFFFFFFFF, message);
	return 1;
}
