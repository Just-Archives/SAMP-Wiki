Description:

Maakt de world boundries aan voor een speler - spelers kunnen niet uit de boundries.


Parameters:
(playerid,Float:x_max,Float:x_min,Float:y_max,Float:y_min)
playeridHet ID van de speler waarvan je de world boundries aan wilt maken.
Float:x_maxDe maximale X coördinaat waar de speler heen mag.
Float:x_minDe minimale X coördinaat waar de speler heen mag.
Float:y_maxDe maximale Y coördinaat waar de speler heen mag.
Float:y_minDe minimale Y coördinaat waar de speler heen mag.


Return Values:

This function does not return any specific values.


Note

Je kan de world boundries voor een speler resetten door de parameters naar 20000.0000, -20000.0000, 20000.0000, -20000.0000 te zetten. 


public OnPlayerSpawn(playerid)
{
    SetPlayerWorldBounds(playerid, 20.0, 0.0, 20.0, 0.0);
    return 1;
}
