This NPC callback was added in SA-MP 0.3a and will not work in earlier versions!


Description:

Este callback es llamado cuando un NPC se conecta al servidor.


Nota

Para la version de un jugador (no NPC) mirar OnPlayerConnect.


Parámetros:
(myplayerid)
myplayeridID del NPC conectado.


Ejemplo: 

public OnNPCConnect(myplayerid)
{
    printf("Se conecto un nuevo NPC: %i", myplayerid);
}

Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • NPC:OnNPCDisconnect Es: Llamado cuando un NPC se desconecta del servidor.
  • OnPlayerConnect Es: Llamado cuando un jugador se conecta al servidor.
