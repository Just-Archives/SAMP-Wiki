Skins Added in 0.3d RC5

This page contains a collection of the skins added in 0.3d RC5. These skins were added thanks to the forum user IllidanS4, having provided the IDs and the voice file names. Each skin has information about the town of origin, ID number and the short model name.

These skins (3, 4, 5, 6, 8, 42, 65, 86, 119, 149, 208, 273, 289) were added in SA-MP 0.3d RC5 and will not work in earlier versions!
