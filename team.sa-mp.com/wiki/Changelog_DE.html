Aktuelle Version

[edit]

Version 0.3x

Erscheinungsdatum: 31.01.2013

  • Ein neues Modell-Vorschau-System erlaubt Servern, 2D Bild von Skins, Fahrzeugen und Objekten (damit auch Waffen) auf dem Bildschirm anzuzeigen
  • Die Sichtweite von Objekten kann nun beliebig weit gesetzt werden (mit CreateObject), was "Pop-In" Effekte in eigenen Gebieten vermindert.
  • Objekt-Material und -Farb-Änderungen wurden verbessert.
  • Eine verbesserte, synchrone Animation für Spieler, die Objekte tragen (SPECIAL_ACTION_CARRY).
  • Friendly-Fire System für Fahrzeuge, wenn die Teamfunktionen verwendet werden (EnableVehicleFriendlyFire).
  • Einige neue Hut- und Haarobjekte hinzugefügt (Hier).
  • Viele wichtige Änderungen an SA:MP's Netzwerkcode, die Netzwerk-Überlastungen reduzieren und die Sicherheit für Serverbetreiber erhöhen.
  • Ein Sanduhr-Icon wird neben den Namen von Spielern, die im Pausemenü oder auf dem Desktop sind, angezeigt (kann mit /nametagstatus deaktiviert werden).
  • Fahrzeugfarben von 128-255 wurden mit einigen exotischeren Farben ersetzt. Einige Farben in diesem Bereich waren zuvor verwendbar, andere waren aber meistens zufällig oder sahen gleich aus. Diese ganzen Farben wurden überarbeitet.

Vorherige Versionen

[edit]

Version 0.3e

Erscheinungsdatum: 08.05.2012

  • Viele kleine Bugfixes, die Sicherheitsupdates beinhalten.
  • Viele neue Objekte zum Mappen, wie z.B. Wände und begehbare Häuser.
  • Texturen und Farben von Objekten können jetzt geändert werden.
  • Neue Funktionen für mehr Kontrolle über die Spielerkamera.
  • Neue Hilfsmittel machen es einfacher, an Spieler befestigte Objekte zu bearbeiten.
  • Neue Animation (+ Handschellen-Objekt) für Verhaftungen.
  • Textfunktionen, um eigene Schriftzüge erstellen zu können.
  • Die Maus kann nun aktiviert werden, um Textdraws/Bilder anklicken zu können.
  • Neues Objektlimit von 1000.


[edit]

Version 0.3d

Erscheinungsdatum: 01.12.2011

  • Unterstützung von Shoutcast/Icecast Musikstreams.
  • Generelle Performance-Verbesserungen und sanfteres Game-play.
  • Synchronisation verbessert, sowie das Einsteigen in Fahrzeuge und auf Motorräder.
  • Neue Objekte, wie z.B. ein Ersatz-Einkaufszentrum, Heißluftballons und Feuerwehr-Hüte.
  • Man kann auf Objekten, die an sich bewegenden Objekten oder an Fahrzeugen befestigt sind, surfen.
  • Statische (auf der Map platzierte Standard-)Objekte können entfernt werden.
  • Sich bewegende Objekte können nun weich rotieren.
  • Viele neue Scripting-Funktionen und -Callbacks für SA:MP Server.


[edit]

Version 0.3c

Erscheinungsdatum: 29.12.2010

  • Ein neu gestaltetes Fahrzeug-Nummernschild-System.
  • Partikelobjekte für Effekte wie z.B. Feuer.
  • Enthält neue und überarbeitete Objekte wie Hüte und Brillen, die getragen werden können.
  • Enthält neue Stunt-Objekte und -Gebäude.
  • Verbessertes Fahrzeug-/Objekt-Surfing.
  • Fahrzeugmotoren und -lichter können nun vom Server gesteuert werden.
  • Man kann nun mit fast jeder Waffe Passagier-Drive-by's ausüben.
  • Schriftfarben-Änderungen im Chat und in Dialogboxen.
  • Ein paar Scripting-Verbesserungen und Bugfixes.


[edit]

Version 0.3b

Erscheinungsdatum: 21.08.2010

Client:

  • "/headmove"-Befehl hinzugefügt, um Kopfbewegungen von Spielern lokal zu de-/aktivieren.
  • Negative Punktzahlen von Spielern werden im Browser als 0 angezeigt.
  • Die Fähigkeit, Objekte wie Fernseher und Stereoanlagen in Häusern aufzuheben, wurde entfernt.
  • Probleme mit Vektoren beim Zielen mit Waffen in verschiedenen Bildschirm-Auflösungen wurden behoben.
  • Probleme mit Spieler-3D-Texten, die nach einem Gamemode-Neustart nicht angezeigt wurden, behoben.
  • Crash, der beim Verlassen des Spiels, während man einen Spieler beobachtet, auftrat, behoben.
  • Fehler im Browser bei samp:// URLs behoben, der auftrat, wenn der Port 5 Zeichen lang war.
  • Namensschild-Rendering in RC-Fahrzeugen korrigiert.
  • Fehler mit dem Pickup-Streaming behoben, der beim Spielertod auftrat.
  • Kopfbewegungen in Fahrzeugen hinzugefügt.
  • Chatlogging eingeführt.
  • Spieler- und Animationssynchronisation verbessert.
  • Fahrzeugsynchronisation verbessert.
  • Dateien, inklusive Screenshots, werden nun in "/Eigene Dokumente/GTA San Andreas User Files" gespeichert.
  • Keine Beschränkung für Bildschirm-Auflösungen.
  • Alle Frame-Limiter Änderungen werden nun gespeichert.
  • Chat-Zeiten mit dem "/timestamp"-Befehl.

Server:

  • forcesync (0-2) Option für ApplyAnimation.
  • SQLite Version auf 3.7.0.1 geupdated.
  • Das Server-Passwort kann mit "password 0" durch die Konsole zurückgesetzt werden.
  • Verbesserungen für GetPlayerWeaponData und GetPlayerAmmo.
  • Neue Fahrzeug-Geschwindgkeits- und -rotationsfunktionen.
  • Mit GetPlayerAnimationIndex können alle abgespielten Animationen überwacht werden.
  • Objekte können an Spielern angebracht werden.
  • Ein Prozessbasierter HTTP Client für Pawn (Beta).
  • Neues Objektlimit von 400.
  • Möglichkeit, die Sichtweite von Objekten zu ändern (bis zu 300 Einheiten).
  • NPCs können nun Animationen aufnehmen.
  • PVar-Geschwindigkeit erhöht.


[edit]

Version 0.3a

Erscheinungsdatum: 17.10.2009

  • Bis zu 500 Spieler pro Server und bis zu 2000 Fahrzeuge.
  • Neue, mit der Maus bediente Punktetafel, Chat und andere UIs.
  • Sichtbare Fahrzeugschäden und Fahrzeug-Reparaturen.
  • NPC Bots, die Züge fahren, Flugzeuge fliegen und einiges mehr können.
  • Komplettes Serverseitiges Game-play.
  • Synchronisation und Fahrzeugsurfing verbessert.
  • Viele neue Scripting-Funktionen.
  • Viele Bug- und Crashfixes - SA:MP 0.3a ist die bisher stabilste Version!


[edit]

Version 0.2X

Erscheinungsdatum: 20.11.2008

  • Unsichere Verarbeitung von Privaten Nachrichten behoben.
  • Paketfragmentierung/MTU Problem behoben.
  • Einige Sicherheitsupdates für den Netzcode und die Verbindungslogik.
  • Sicherheitsupdates für den Client.
  • Kompression der Vektoren beim Zielen entfernt, um die Genauigkeit zu erhöhen.
  • Sekunden zum Server Log hinzugefügt, es zeigt nun auch rohe, eingehende UDP-Verbindungen an.
  • OnVehicleMod/OnVehiclePaintjob/OnVehicleRespray enthalten nun die Spieler ID.
  • OnPlayerUpdate hinzugefügt, ein Callback, der bei jedem Spieler Netzwerk-Update abgerufen wird. Das wird serverseitigen Anti-Cheat Systemen helfen.
  • Anzahl der Filterscripts auf 16 beschränkt.

Version 0.2.2 R3 (Nur Client)

Erscheinungsdatum: 20.06.2008

  • SAC Code vollständig entfernt. Das behebt ein Sicherheitsproblem, das schon früher angesprochen wurde.
  • Ein Problem mit dem Kopieren von Text in die Zwischenablage behoben.
  • Zeit-/Nebel-/Beleuchtungsflackern behoben, wenn der Verlauf der Zeit deaktiviert ist.
  • Einige Fahrzeug-Audi Probleme behoben (Stille Autos/Flugzeuge/Boote/Helis).
  • Texturenlag-/Streaming-Popups wenn viele Modelle geladen werden, behoben.
  • Einige Tuningshop Crashes behoben.
  • Die alternative Sirene von Polizeifahrzeugen ist nun synchron.
  • Der ganze TextDraw wurde neu geschrieben. Sie funktionieren nun in Innenräumen und werden ausgeblendet, wenn die Punktetafel angezeigt wird.
  • Crashes, die von DestroyVehicle verursacht wurden, wurden behoben.
  • Etwas vom 0.3/0.2.5 Fahrzeugsurfing Code importiert.
  • "/fpslimit"-Befehl hinzugefügt, um den Frame-Limiter einzustellen. Erlaubte Raten liegen zwischen 20 und 1000.

Version 0.2.2 R2 (Nur Client)

Erscheinungsdatum: 30.08.2007

  • Einige Crashes, die von Modellen/Pickups hervorgerufen wurden, wurden behoben.
  • Kollisions-Verarbeitungs-Probleme behoben (Objekte/Spieler fallen durch den Boden).
  • Das Chat-Fenster verlegt Text aus zu langen Zeilen in die nächste Zeile (Probleme behoben).
  • Texturen-Rendering behoben, wenn deine Grafikkarte eine eigene d3dx9_25.dll verwendet.


[edit]

Version 0.2.2

Erscheinungsdatum: 29.08.2007

  • Das Chat-Fenster kann nun gescrollt werden mit bis zu 5 Seiten (50 Zeilen).
  • Das Chat-Fenster verlegt Text aus zu langen Zeilen nun sauber in die nächste Zeile.
  • Chat-Einträge verbessert. Sie erlauben nun bis zu 128-Zeichen-Nachrichten, Unterstützen das Markieren von Text, Kopieren und Einfügen usw.
  • Das Text-Rendering wurde schwer optimiert. Der Chat hat nun keinen Einfluss mehr auf die Framerate.
  • Namen können nun durch Wände/Objekte gesehen werden.
  • Ein Tuningshop-Crash, der von bestimmen Tuningteilen ausging, wurde behoben.
  • Das Pickup-Limit wurde auf 400 erhöht.
  • Der Server kann nun alle Ein- und Ausgänge aus dem Einzelspieler entfernen.
  • Nativer SQLite-Datenbank-Support für Pawn. SQLite ist eine leichter Alternative gegenüber dem MySQL Plugin.
  • Die Distanz, aus der Namensschilder gesehen werden können, kann nun vom Server bestimmt werden.
  • Unglaublicher-Stunt-Boni können nun vom Server ein- und ausgeschaltet werden.
  • Einige Netzcode-Sicherheitsupdates.
  • 0.2.2 wird die zweite Hauptversion von SAC verwenden.
  • Spieler-Kopfbewegungen sind nun synchron. Der Character wird in die Richtung sehen, in die die Kamera sieht.
  • Eine neue Telefon-Aktion erlaubt Spielern, ein Telefon zu halten und auszusehen, als würden sie telefonieren.
  • Neue verwendbare Skins: Tenpenny, Pulaski, Hernandez, Ryder, Smoke, Sweet, Mafia Boss.
  • Bandbreiten- und Spieler-Character-Synchronisation verbesert. 0.2.2 wird weniger Server-Bandbreite verbrauchen als 0.2.1.
  • Ein Bug beim Setzen der Farben von Spielern ab ID 100 wurde behoben.
  • Timeout-Zeiten erhöht, um Verbindungs-Abbrüche zu verhindern.
  • CPU-Zeit, um Fahrzeuge zu verarbeiten, verringert.
  • Animationsprobleme mit Hydranter und der Spraydose behoben.
  • Namensschilder ohne Spieler behoben.
  • Problem behoben, bei dem dem Spieler die "You are banned" und "Server is full"-Nachrichten nicht angezeigt wurden.
