Client

[edit]

Ho un'errore "San Andreas non può essere trovato"

San Andreas Multi-player non è un programma fisso! Si aggiunge la funzionalità multi-player per San Andreas, e quindi hai bisogno di GTA San Andreas per PC - inoltre devi avere EU/US v1.0, altre versioni, come v2.0 o Steam e versioni Direct2Drive non funzioneranno.

[edit]

Non riesco a vedere nessun server in SA-MP browser

Prima di tutto, garantire la sta seguendo le procedure indicate nella Piccola Guida. Se avete seguito quello che dice, e ancora non vedo alcun server, è necessario consentire SA:MP accesso attraverso il firewall. Purtroppo, a causa della grande quantità di firewall disponibili, non possiamo offrire un ulteriore sostegno su questo - noi suggeriamo di consultare il sito web dei produttori o fare una ricerca su Google. Assicuratevi inoltre di avere l'ultima SA: MP versione!

[edit]

Carica SinglePlayer invece di SA:MP

NON DEVI ACCERDERE AGLI OZIONI DI SINGLEPLAYER COME (NEW GAME, LOAD GAME, ECC) - SA:MP SI CARICHERà DA SOLO E NON AVRà BISOGNO DI ALUCN'OPZIONE. SE VEDI "NEW GAME" SI CARICHERà SINGLEPLAYER, NON SAN ANDREAS MULTIPLAYER.

Si può caricare in single player per 2 motivi - hai installato SA:MP nella cartella sbagliata o hai la versione sbagliata di San Andreas. Se hai la versione sbagliata è facile da risolvere. Scarica la patch che la downgrada nella versione giusta. Clicca qui per scaricare la patch.

[edit]

Mi viene scritto "Unacceptable NickName" mentre mi collego su un server

Assicurarsi che non utilizzi alcun caratteri non ammessi nel tuo nome (usa solo 0-9, a-z, [], (), $, @, ., = ), e il tuo nickname non deve essere più lungo di 24 carrateri. Questo potrebbe anche essere causato quando un giocatore è su un server con lo stesso nome al tuo o se si tenta di connettersi al server subito dopo un crash.

[edit]

Screen sticks at "Connecting to ip:port..."

Il server potrebbe essere non in linea, o se non puoi collegare a qualsiasi server, disattivare il firewall e vedere se funziona. In caso contrario è necessario configurare il firewall correttamente - Vai al suo sito web per scoprire come. Potrebbe anche essere che hai una vecchia versione di SA-MP, scaricare l'ultima versione dal SA-MP download page.

[edit]

Ho modificato GTA: San Andreas, e il mio SA:MP non si carica

Se non riesce a caricarsi si prega di cancellare le mod. Ulteriori informazioni sulle mod e perché può causare problemi con il gioco può essere trovato qui

[edit]

Quando apro GTA con SA:MP non si avvia

Cancellare il file gta_sa.set dalla cartella userfiles e assicurati di non avere cheats/trucchi/mod.

[edit]

Server

[edit]

Il server crasha appena mi collego

Gli errori più comuni si trova nel file server.cfg. Controlla server_log.txt o la ragione che causa nel file crashinfo.txt.

[edit]

Il mio server non funziona Ho provato a disattivare il firewall, ma ancora funziona ancora

Dovete fare il portforward alle vostre porte! Puoi fare questo tramite PF Port Checker Scaricala da: www.portforward.com Se le porte non vengono forwardate significa che devi aprire le porte nel router. Controlla la lista dei router http://portforward.com/english/routers/port_forwarding/routerindex.htm Ci sono tutte le informazioni su come aprire le porte.

[edit]

I giocatori costantemente ricevendo l'errore "Unacceptable NickName" ma è valido

Se sei sicuro di utilizzare un nome accettabile e il server gira su Windows quindi prova di modificare l'opzione di compatibilità del server.exe samp-a Windows 98 e dovrebbe essere fissato dopo un riavvio del server.

I server nella versione Windows se sono aperti per troppo tempo può causare blocchi. Questo è stato notato di circa 50 giorni di uptime del server. Per risolverlo, si richiede un riavvio.
