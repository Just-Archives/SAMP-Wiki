Db open

Beschreibung:

Diese Funktion wird benutzt, um eine SQLite database aus dem /scriptfiles Ordner zu öffnen


Parameter:
(name[])
name[]Der Name der database

Return Values:

Gibt die ID der database zurück


Beispiel:

new DB:USERDB = db_open("users.db");


public OnGameModeInit( )
{
    new DB:USERDB;
    USERDB = db_open("users.db");
    return 1;
}
[edit]

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • db_close: Schließt eine SQLite database
