Template:DisplayTitle

From SA-MP Wiki

Jump to: navigation, search

Documentation

This template can be used to change the shown title on a page. This is useful for pages which either need to start with a lowercase letter, or that have underscores.

It can be also used on all translated articles, to remove the language code from the title (see example below).

Note:

  • On editing preview, this template will also replace the "Editing" or "Edit Section" text, but it should be ignored and you should know that you are editing the page.
  • The title should be same with the page subject.

Syntax

{{DisplayTitle|title_to_display_here}}

Example

Lowercase function with an underscore, db_close: 

{{DisplayTitle|db_close}}

Function in another article language to remove the language suffix, so only the function is displayed on title.
Example: 'SetPlayerPos_RU' to 'SetPlayerPos' by adding the code in SetPlayerPos_RU, 

{{DisplayTitle|SetPlayerPos}}
