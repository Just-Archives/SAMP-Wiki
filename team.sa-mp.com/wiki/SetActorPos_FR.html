Cette function a été ajoutée dans SA-MP 0.3.7 et ne fonctionnera pas sur des versions antérieures!


Description:

Modifie la position d'un actor.


Parameters:
(actorid, Float:X, Float:Y, Float:Z)
actoridID de l'actor dont modifier la position. Retourné par CreateActor.
XNouvelle coordonnée X de l'actor.
YNouvelle coordonnée Y de l'actor.
ZNouvelle coordonnée Z de l'actor.


Retourne:

  • 1: La fonction a bien été exécutée.
  • 0: La fonction n'a pas pu être exécutée. L'actor spécifié n'existe pas.


Note

Lorsque vous créez un actor avec CreateActor, vous spécifiez déjà sa position. Vous n'avez pas besoin d'utiliser cette fonction sauf si vous voulez modifier sa position plus tard. 


new MyActor;
 
public OnGameModeInit()
{
    MyActor = CreateActor(...);
    return 1;
}
 
// Quelque part dans votre code
SetActorPos(MyActor, 1.0, 2.0, 3.0);
[edit]

Fonctions Relatives

Les fonctions suivantes peuvent être utiles car elles sont indirectement ou directement liées a cette fonction.

  • GetActorPos: obtenir la position d'un actor.
  • CreateActor: créer un actor.
