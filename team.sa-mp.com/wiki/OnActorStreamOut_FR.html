Cette callback a été ajoutée dans SA-MP 0.3.7 et ne fonctionnera pas sur des versions antérieures!


Description:

Cette fonction est appelée quand un actor est déchargé (n'est plus visible) pour un joueur.


Paramètres:
(actorid, forplayerid)
actoridL'ID de l'actor déchargé par le joueur.
forplayeridL'ID du joueur qui a déchargé l'actor.


Retourne:

Cette callback ne retourne rien, mais doit retourner quelque chose! Regardez ICI pour plus d'informations.


public OnActorStreamOut(actorid, forplayerid)
{
    static
        string[40];
 
    format(string, sizeof(string), "L'actor ID %d vient d'être déchargé de votre client.", actorid);
    SendClientMessage(forplayerid, 0xFFFFFFFF, string);
 
    return 1;
}
[edit]

Callback Relatives

Les Callbacks ci dessous sont indirectement ou directement liées à cette Callback.

  • OnPlayerStreamIn: Quand un joueur est chargé (devient visible) pour un autre joueur
  • OnPlayerStreamOut: Quand un joueur est dé-chargé (devient invisible) pour un autre joueur
  • OnActorStreamIn: Quand un actor est chargé (devient visible) pour un joueur.
  • OnVehicleStreamIn: Quand un véhicule est chargé (devient visible) pour un joueur
  • OnVehicleStreamOut: Quand un véhicule est dé-chargé (devient invisible) pour un joueur
