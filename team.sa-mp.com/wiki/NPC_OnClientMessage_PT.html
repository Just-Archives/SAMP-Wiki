NPC:OnClientMessage

NPC Callback foi adicionada na versão SA-MP 0.3a e não irá funcionar em versões antigas!


Descrição:

Esta callback é chamada quando um NPC vê uma ClientMessage. Esta vai callback ser chamada sempre que a função SendClientMessageToAll_PT é usada e sempre que a função SendClientMessage_PT é enviada para um NPC. Esta callback não vai ser chamada quando alguém diz algo. Para uma versão disto mas com o texto do jogador, veja NPC:OnPlayerText_PT.


Parâmetros:
(color, text[])
colorA cor da ClientMessage.
text[]A mensagem.


Valores de retorno:

Esta callback não lida com retornos.


Exemplo: 

public OnClientMessage(color, text[])
{
    if(strfind(text,"Balanço bancário: $0") != -1) SendChat("Sou pobre :( :(");
}
[edit]

Funções Relacionadas

As seguintes funções podem ser úteis, pois estão de alguma forma, relacionadas a esta função.

  • NPC:OnPlayerText: Chamada quando um player ou NPC envia uma mensagem para o chat.
