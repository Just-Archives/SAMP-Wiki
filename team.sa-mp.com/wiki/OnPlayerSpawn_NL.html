OnPlayerSpawn

Beschrijving:

Deze callback wordt opgeroepen wanneer een speler spawnt. (v.b.: nadat de SpawnPlayer functie wordt gebruikt)


Parameters:
(playerid)
playeridHet ID van de speler dat spawnt.


Retourneert:

Returning 0 in deze callback zal de speler terug naar de class selectie forceren wanneer ze de volgende keren spawnen.


Voorbeeld: 

public OnPlayerSpawn(playerid) {
 
	new
		PlayerName[MAX_PLAYER_NAME], // MAX_PLAYER_NAME = 24 + \0
		string[40];
 
	GetPlayerName(playerid, PlayerName, sizeof(PlayerName));
	format(string, sizeof(string), "%s has spawned successfully.", PlayerName);
	SendClientMessageToAll(0xFFFFFFFF, string);
	return 1;
}
[edit]

Extra Callbacks

De volgende callbacks kunnen handig zijn aangezien ze op een manier erbij horen.

  • OnPlayerDeath: Opgeroepen wanneer een speler sterft.
  • OnVehicleSpawn: Opgeroepen wanneer een voertuig spawnt.


[edit]

Extra Functies

De volgende functies zijn mogelijk ook bruikbaar, aangezien ze op een manier erbij horen.

  • SpawnPlayer: Spawnt de speler.
  • AddPlayerClass: Voegt een nieuwe class toe aan de class selectie.
  • SetSpawnInfo: Spawnt een speler met extra gegevens.
