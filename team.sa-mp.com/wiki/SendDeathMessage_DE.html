Description:

Fügt einen Tod zum "'killfeed'" auf der rechten Seite des Bildschirms (siehe Bild unten) hinzu.


Parameter:
(killer, killee, weapon)
killerDie ID des Killers (kann INVALID_PLAYER_ID sein).
killeeDie ID des Spielers, der gestorben ist.
weaponDie Todesursache (nicht immer eine Waffe). Es können auch spezielle Symbole verwendet werden (wie ICON_CONNECT und ICON_DISCONNECT).


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


public OnPlayerDeath(playerid, killerid, reason)
{
    SendDeathMessage(killerid, playerid, reason);
    return 1;
}
[edit]

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • SendClientMessage: Sendet eine Nachricht an einen bestimmten Spieler.
  • SendClientMessageToAll: Sendet eine Nachricht an alle Spieler.


[edit]

Zugehörige Callbacks

Die folgenden Callbacks könnten hilfreich sein, da sie zusammen mit der Funktion arbeiten oder die Funktion dort ausgeführt werden kann.

  • OnPlayerDeath: Wird aufgerufen, wenn ein Spieler stribt.
