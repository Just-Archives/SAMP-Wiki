For some reason external links don't work in templates?

I tried doing "tip|This function can be used to retrieve a vehicle's speed (km/h, m/s or mph). For more info look here." But it simply shows up as

Tip

{{{1}}}


Any known fix for this?

Drebin 02:20, 4 September 2013 (UTC)


Tip

This function can be used to retrieve a vehicle's speed (km/h, m/s or mph). For more info look here.

It's because of the equal sign. Just use numbered parameters. Blantas 12:54, 4 September 2013 (UTC)


Awesome, thanks Blantas!
