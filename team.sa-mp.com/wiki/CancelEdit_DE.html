CancelEdit

Diese Funktion/Dieses Callback wurde erstmals in SA:MP 0.3e hinzugefügt und funktioniert nicht in älteren Versionen!


Beschreibung:

Beendet den Objektbearbeitungsmodus für einen Spieler.


Parameter:
(playerid)
playeridDie ID des Spielers, dessen Objektbearbeitung beendet werden soll.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


Beispiel: 

public OnPlayerCommandText(playerid, cmdtext[])
{
    if(!strcmp(cmdtext, "/stopedit", true))
    {
        CancelEdit(playerid);
        SendClientMessage(playerid, 0xFFFFFFFF, "SERVER: Du hast mit dem Bearbeiten aufgehört.");
        return 1;
    }
    return 0;
}
[edit]

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • SelectObject: Lässt einen Spieler ein Objekt auswählen.
  • EditObject: Lässt einen Spieler ein Objekt bearbeiten.
  • EditPlayerObject: Lässt einen Spieler ein spielerseitig erstelltes Objekt bearbeiten.
  • EditAttachedObject: Lässt einen Spieler ein Attached-Objekt bearbeiten.


  • CreateObject: Erstellt ein Object.
  • DestroyObject: Löscht ein Objekt.
  • MoveObject: Bewegt ein Objekt.
