Description:

Fügt ein Eintrag zum angegebenen Menü hinzu.



Wichtiger
Hinweis

Es können nur 12 Einträge in einem Menü erstellt werden.


Hinweis

Es können 2 Spalten erstellt werden, Spalte 0 und 1



Parameter:
(Menu:menuid, column, menutext[])
menuidID des Menüs, dem ein Eintrag hinzugefügt werden soll.
columnDie Spaltennummer des Eintrags
menutext[]Der Titel des neuen Eintrags.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


new Menu:examplemenu;
// Globale Deklaration des Menüs
// für die spätere Verwendung
 
public OnGameModeInit()
{
    examplemenu = CreateMenu("Dein Menü", 2, 200.0, 100.0, 150.0, 150.0);
    // erstellt ein Menü mit 2 Spalten an Position 200 x 100
    // mit der Breite von 150
    AddMenuItem(examplemenu, 0, "Eintrag 1");
    AddMenuItem(examplemenu, 1, "Eintrag 2");
    //erstellt 2 Einträge im Menü
    //bei ungleicher Spalten ID werden die Einträge nebeneinander dargestellt
    //bei gleicher Spalten ID werden die Einträge untereinander dargestellt
    return 1;
}


Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • CreateMenu: Erstellt ein Menü.
  • SetMenuColumnHeader: Legt die Spaltenbeschriftung fest.
  • DestroyMenu: Löscht ein erstelltes Menü
  • ShowMenuForPlayer: Zeigt einem Spieler das erstellte Menü an
  • HideMenuForPlayer: Lässt das Menü für einen Spieler verschwinden.
  • GetPlayerMenu: Gibt die ID des Menüs zurück, welches sich auf dem Spielerbildschrim befindet.


Zugehörige Callbacks

Die folgenden Callbacks könnten hilfreich sein, da sie zusammen mit der Funktion arbeiten oder die Funktion dort ausgeführt werden kann.

  • OnPlayerSelectedMenuRow: Startet wenn ein Spieler ein Menueintrag auswählt.
  • OnPlayerExitedMenu: Startet wenn ein Spieler ein Menu verläst.
