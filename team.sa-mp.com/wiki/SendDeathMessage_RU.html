SendDeathMessage

5 самых последних убийств показаны игрокам на правой стороне экрана. Нажав F9 можно скрыть/показать килл лист.

Описание:

Добавляет смерть в 'килл лист' на правой стороне экрана всем игрокам.


Параметры:
(killer, killee, weapon)
killerID убийцы (может быть INVALID_PLAYER_ID).
killeeID умершего игрока.
weaponПричина (не всегда оружие), по которой умер игрок. Также возможно использование специальных иконок (ICON_CONNECT и ICON_DISCONNECT).


Возвращаемые значения:

Эта функция всегда возвращает 1, даже если ей не удалось выполниться. Такое может произойти (нет сообщения), если 'playerid' не существует. Если 'reason' указан неверно - будет показана иконка 'череп-и-кости'. 'killerid' разрешается указывать несуществующим (INVALID_PLAYER_ID).


Пример использования: 

public OnPlayerDeath(playerid, killerid, reason)
{
    SendDeathMessage(killerid, playerid, reason);
    return 1;
}
Советы
  • Сообщения о смертях могут быть очищены, если указать допустимый ID для 'playerid', но который не подключен.
  • Чтобы показать сообщение для конкретного игрока, используйте SendDeathMessageToPlayer.
  • Вы можете использовать NPC для создания собственных причин смерти.


[edit]

Связанные функции

Следующие функции могут быть полезны, т.к. они так или иначе связаны с текущей функцией.

  • SendClientMessage: Send a message to a certain player.
  • SendClientMessageToAll: Send a message to all players.
  • SendDeathMessageToPlayer: Add a kill to the death list for a player.
