Description:

Adiciona um veículo fixo.


Nota

Só pode ser usado em OnGameModeInit


Parâmetros:
(modelid, Float:spawn_x, Float:spawn_y, Float:spawn_z, Float:angle, color1, color2)
modelidModelo do veículo.
Float:spawn_XCoordenada-X para o veículo.
Float:spawn_YCoordenada-Y para o veículo.
Float:spawn_ZCoordenada-Z para o veículo.
Float:angleDireção do veículo - Angulo.
color1Cor Primária.
color2Cor Secundária.


Valores de retorno:

  • O ID do veículo criado (1 - MAX_VEHICLES).
  • INVALID_VEHICLE_ID (65535) Caso o veículo não tenha sido criado(Limite de veículos alcançado ou modelo invalido).


public OnGameModeInit( )
{
    // Add um Hydra ao jogo
    AddStaticVehicle(520, 2109.1763, 1503.0453, 32.2887, 82.2873, 0, 1);
 
    return 1;
}
