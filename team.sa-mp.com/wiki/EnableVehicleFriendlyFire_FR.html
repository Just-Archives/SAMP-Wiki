Cette fonction a été ajoutée dans SA-MP 0.3x et ne marchera pas dans les versions antérieures


Description:

Active le tir sur les véhicules alliés. Les joueurs vont être autorisés à tirer sur les véhicules dont le conducteur est quelqu'un de la même équipe que lui.


Cette fonction n'a pas de paramètres.


RetourneCette fonction ne retourne pas de valeur spécifique.


public OnGameModeInit()
{
    EnableVehicleFriendlyFire();
    return 1;
}
