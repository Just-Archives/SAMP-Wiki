SetPlayerSkin

Die "SetPlayerSkin" Funktion kann den Skin eines Spielers in einem anderen verwandeln. Diese Funktion kann jeden moment benutzt werden.

ACHTUNG: Die folgenden Skin ID's crashen das Spiel: 3, 4, 5, 6, 7, 8, 42, 65, 74, 86, 119, 149, 208, 268, 273, 289.


Parameters:
(playerid, skinid)
playerid Die Spieler ID, die du willst, die den Skin ändert.
skinDen Skin, den den Spieler will.


public OnPlayerCommandText( playerid, cmdtext[] )
{
    if( strcmp( cmdtext, "/meinskin", true) == 0)
    {
        // Setzt den Spielerskin auf ID 277, einen Feuerwehrmann. Nicht alle Skins
        // können hier benutzt werden.
        SetPlayerSkin(playerid, 277);
        return 1;
    }
    return 0;
}

Ähnliche Funktionen:

  • SetSpawnInfo: Diese Funktion kann den Skin ändern oder andere Infos beinhalten.
  • SetPlayerTeam: Updatet das Spielerteam um in einem anderen zu sein, nützlich für TDM Gamemodes.
  • SpawnPlayer: Diese Funktion re-spawnt den Spieler.
