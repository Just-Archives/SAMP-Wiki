Description:

Bestimmt, ob die Verwendung von Waffen in Innenräumen erlaubt ist, oder nicht.


Parameter:
(allow)
allow1 erlaubt Waffen in Innenräumen, 0 verbietet sie.


RückgabeDiese Funktion gibt keinen relevanten Wert zurück.


public OnGameModeInit()
{
    // Waffen in Innenräumen erlauben.
    AllowInteriorWeapons(1);
    return 1;
}

Verwandte Funktionen

Folgende Funktionen könnten ebenfalls nützlich sein, da sie in der einen oder anderen Weise mit dieser verwandt sind.

  • SetPlayerInterior: Besimmt die Interior ID für einen Spieler.
  • GetPlayerInterior: Gibt die ID des Interiors eines Spielers zurück.
  • SetPlayerArmedWeapon: Gibt einem Spieler die angegebene Waffe in die Hände (sofern vorhanden).

Zugehörige Callbacks

Die folgenden Callbacks könnten hilfreich sein, da sie zusammen mit der Funktion arbeiten oder die Funktion dort ausgeführt werden kann.

  • OnPlayerInteriorChange: Wird aufgerufen, wenn ein Spieler den Innenraum wechselt.
