¡Este/a callback fue agregado/a en SA-MP 0.3a y no funcionará en versiones anteriores!


Descripción:

Este callback es llamado cuando un vehiculo es "descargado" para un jugador/cliente.


Parámetros:
(vehicleid, forplayerid)
vehicleidID del vehiculo que es "descargado".
forplayeridID del jugador al cual se le "descarga" el vehiculo.


public OnVehicleStreamOut(vehicleid, forplayerid)
{
    new string[36];
    format(string, , "El vehiculo %i te fue \"descargado\".", vehicleid);
    SendClientMessage(forplayerid, 0xFFFF00FF, string);
    return 1;
}

Callbacks relacionadas

Las siguientes callbacks pueden ser útiles, ya que éstas están relacionadas con esta función de alguna u otra manera.

  • OnPlayerStreamIn ES: Llamado cuando un jugador es "cargado" para otro jugador.
  • OnPlayerStreamOut ES: Llamado cuando un jugador es "descargado" para otro jugador.
  • OnVehicleStreamIn ES: Llamado cuando un vehiculo es "cargado" para un jugador.
