爆炸列表

用于 CreateExplosion 。

类型 可见 分裂 火苗 物理爆炸 特殊效果 范围
0 是 否 无 是 无 大
1 是 否 有 否 无 一般
2 是 否 有 是 无 大
3 是 否 有时候 是 无 大
4 是 是 无 是 Unusual explosion, produces just special blast burn FX effects and blasts things away, NO SOUND EFFECTS. 一般
5 是 是 无 是 Unusual explosion, produces just special blast burn FX effects and blasts things away, NO SOUND EFFECTS. 一般
6 是 否 无 是 额外的红色爆炸余辉 非常大
7 是 否 无 是 额外的红色爆炸余辉 巨大
8 否 否 无 是 不可见 一般
9 否 否 有 是 在地平面上创建火苗，此外可听见爆炸声但爆炸不可见 一般
10 是 否 无 是 无 大
11 是 否 无 是 无 小
12 是 否 无 是 很小 非常小
13 否 否 无 否 不产生地上的黑色烧痕以外的其他特殊效果，也没有伤害 大
