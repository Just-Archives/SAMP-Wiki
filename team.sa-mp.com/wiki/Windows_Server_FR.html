[edit]

Nécessaire a la création du serveur

  • Un accès internet (un serveur de 100 joueurs utilise 4Mb par seconde)
  • Un port UDP ouvert (7777 par défaut) | Guide
  • Une adresse IP fixe peut servir mais n'est pas nécessaire
[edit]

Mise en place du serveur

[edit]

Obtenir les fichiers

La première étape pour créer votre serveur SAMP est de télécharger les fichiers du Serveur Windows sur la Page des téléchargements SAMP où vous aller pouvoir trouver la dernière version des fichiers du Serveur Windows Sous l'entête SA-MP Windows Server Download - cliquez sur un des liens pour commencer le téléchargement, soyez sûr de bien mettre les fichiers téléchargés dans un endroit ou vous saurez le retrouver.

[edit]

Configurer le serveur

Une fois que les fichiers seront , extrayez (en utilisant un programme qui peut lire les fichiers .rar ) les fichiers (il n'y a pas besoin de les mettres dans un dossier spécifique, comme le serveur n'a pas besoin de GTA:SA ni de SA:MP). Maintenant, nous allons configurer le server.cfg.

Trouvez le fichier server.cfg dans le dossier dans lequel vous avez mis les fichiers de votre serveur et ouvrez le avec le Bloc-Note ou dans un autre éditeur de texte. Le fichier ouvert devrais apparaitre en Anglais. The fichier est fais pour configurer le serveur, vous pourrez spécifier si le lanmode est on ou off, le Nombre de joueurs, le port du serveur, lenom du serveur, le(s) game modes, les filterscripts*, l'adresse de votre site et votre mot de passe RCON.

[edit]

Lancer le serveur

Une fois que vous avez configuré votre serveur, vous pouvez le lancer. Pour le lancer, il suffit simplement de cliquer sur samp-server.exe et le serveur se lancera.
