Kilens

[edit]

Ezt a hibaüzenetet kapom: "San Andreas nem található"

A San Andreas Multi-player nem egy különálló program! Hozzáad egy működő multiplayert a San Andreashoz, és ezért kell a GTA San Andreas PC-s változata. - ennek is az EU/US v1.0-es verziója kell hogy meglegyen, egyéb verziók, mint a v2.0 vagy Steam és Direct2Drive verziók nem fognak működni.

[edit]

Nem látok egy szervert sem a SA:MP szerverkeresőben

Először is, bizonyosodj meg róla, hogy követted a meghatározott eljárásokat a Quick-start guide-ben. Ha követted, amit mond, és mindig nem látsz szervereket, akkor engedélyezned kell a SA:MP hozzáférését a tűzfaladon. Sajnos a nagy mennyiségű tűzfal szoftverek miatt, nem tudunk további segítséget adni ebben - Unfortunately, due to the large amount of available firewall software, we can not offer further support on this - azt javasoljuk, hogy keresd fel a gyártó oldalát vagy próbálkozz Google keresésekkel. Szintén legyél biztos abban, hogy a legújabb SA:MP verziót használod!

[edit]

Single Player Loads instead of SA:MP

YOU ARE NOT SUPPOSED TO SEE THE SINGLE PLAYER OPTIONS (NEW GAME, LOAD GAME, ETC) - SA:MP SHOULD LOAD BY ITSELF AND NOT PRESENT THESE OPTIONS. IF YOU SEE "NEW GAME" SINGLE PLAYER HAS LOADED, NOT SAN ANDREAS MULTIPLAYER.

Single Player can load for 2 reasons - you have installed SA:MP to the wrong folder or you have the wrong version of San Andreas. If you have the wrong version it's easy to fix. Download this patch that downgrades it to the right version. Click here to download the patch.

[edit]

I get "BAD NICKNAME" when connecting to server

Ensure you are using no disallowed characters in your name (use 0-9, a-z, [], (), $, @, ., = only), and that your name is no longer than 20 characters long. This also could be caused when a player is on a server with the same name as you or if you try connecting to the server immediately after timing out.

[edit]

Screen sticks at "Connecting to ip:port..."

The server could be offline, or if you cant connect to any server, disable your firewall and see if it works. If it does you must set up your firewall properly - go to its website to find out how. It could also be that you have an old version of SA-MP, download the latest version from the SA-MP download page.

[edit]

I have a modified GTA: San Andreas, and my SA:MP won't load

If it wont load then remove your mods. More information why mods can cause issues with your game can be found here

[edit]

When launching GTA with SA:MP it wont start

Delete the gta_sa.set file from your userfiles folder and make sure you don't have any cheats/mods.

[edit]

Server

[edit]

Server instantly crashes when started

Most commonly its a error in your server.cfg file. Check the server_log.txt and the reason should be printed out or in the crashinfo.txt file.

[edit]

Players constantly getting "BAD NICKNAME" error but it is valid

If you are sure you use a right name and the server runs on Windows then try to change the compatibility option of the samp-server.exe to Windows 98 and it should be fixed after a server restart.

Windows servers with high uptime may also cause this issue to occur. This has been noticed of around 50 days of server uptime. To resolve it, a reboot is required.
